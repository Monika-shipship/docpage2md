整理时间：2026-06-12
汇率假设：**1 美元 = 6.78 元人民币**
计价单位：除特别说明外，均为 **每百万 Token** 的输入/输出价格。
使用场景：教材图片、手写笔记、复杂公式、表格、图表识别，并输出 Markdown / LaTeX。

> 说明：阿里云官方价格以中国内地服务部署范围为主。OpenRouter 价格按美元标价折算成人民币。部分阿里云模型存在阶梯计费、Batch 半价、缓存命中折扣、限时折扣；实际账单以调用时平台计价为准。

|---|---|---|

# 8. 按任务场景的最终选择

## 8.1 图片教材、清晰扫描页

首选：`qwen-vl-ocr`
低价试验：`Vanchin/DeepSeek-OCR`
OpenRouter 替代：`qwen/qwen3-vl-32b-instruct`
PDF 批处理：图片合成 PDF 后走 OpenRouter `mistral-ocr`

## 8.2 手写笔记、公式密集页

第一轮：`qwen-vl-ocr` 或 `qwen/qwen3-vl-32b-instruct`
复核：`Qwen3.6-Flash` / `qwen/qwen3-vl-235b-a22b-instruct`
疑难页：`qwen/qwen3-vl-235b-a22b-thinking` / `Gemini 2.5 Pro` / `Qwen3.7-Max-2026-06-08`

## 8.3 图表、实验装置图、物理示意图

低价：`Gemini 2.5 Flash Lite` / `Qwen3.6-Flash`
更强：`Qwen3.7-Plus` / `Gemini 2.5 Flash`
兜底：`Qwen3.7-Max-2026-06-08` / `Kimi K2.6`

## 8.4 Markdown / LaTeX 后处理

主力：`deepseek-v4-flash`
长文档、跨页合并、章节级整理：`deepseek-v4-pro`

# 9. 最简决策树

1. **能用阿里云官方并且是图片 OCR：先用 `qwen-vl-ocr`。**
2. **想极限省钱：先抽样测试 `Vanchin/DeepSeek-OCR`，但注意 8K 限制。**
3. **只能用 OpenRouter：先用 `qwen/qwen3-vl-32b-instruct`。**
4. **Gemini 不建议主力批量 OCR，但 `gemini-2.5-flash-lite` 值得作为对照模型。**
5. **识别结果不要求一次完美，最终交给 DeepSeek V4 Flash/Pro 清洗 Markdown 和 LaTeX。**
