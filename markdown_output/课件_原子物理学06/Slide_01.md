# X射线与Compton效应-X射线的发现

1895年，伦琴发现了X射线

Wilhelm Röntgen (1845-1923)

X射线不带电、很强的穿透性、直线传播、使照相底片感光、使气体电离奇特性等。

The Nobel Prize in Physics 1901

## Figure & Layout Description

幻灯片顶部是主标题，其中"X射线与Compton效应-"为黑色粗体字，"X射线的发现"为红色粗体字，下方有一条深灰色水平分割线。标题下方左侧区域有黑色文字"1895年，伦琴发现了X射线"。中间偏左位置是X射线产生装置的示意图：整体为黑白线条绘制的真空管结构，左侧线圈标注红色字母"C"，右侧倾斜的红色三角形标注"A"，连接外部电路的红色导线标注"Uₕ"（左侧）和"Uₐ"（右侧，含正负极符号）。真空管内部有蓝色区域表示电子束路径，标注"Wᵢₙ"（蓝色箭头向右）和"Wₒᵤₜ"（蓝色箭头向上），绿色虚线箭头从红色三角形向外辐射，标注"X"。右侧垂直排列三张图片：最上方是Wilhelm Röntgen的黑白肖像照（1845-1923）；中间是早期X射线管实物照片（透明玻璃球体连接细长管）；最下方是手部X光照片（可见骨骼结构及手指上的戒指）。底部中央有诺贝尔物理学奖1901的金色奖章图标及文字。所有文字均为黑色，除标题红色部分外，整体背景为纯白色。
