# X射线与Compton效应-X射线的发现

## X光管发射光谱分布

## 轫致辐射(bremsstrahlung，刹车辐射)
$h\nu = T - T'$  
$$h\nu_{\text{max}} = \frac{hc}{\lambda_{\text{min}}} = T = eU$$

杜安(W. Duane)和亨特(P. Hunt)于1915年实验测定了$h$值，与利用光电效应实验测定的$h$值十分接近。

## Figure & Layout Description
页面顶部是主标题"X射线与Compton效应-X射线的发现"，其中"X射线与Compton效应-"为黑色粗体，"X射线的发现"为红色粗体，下方有一条细灰色水平分隔线。标题下方左侧是红色粗体子标题"X光管发射光谱分布"。页面主体分为四个主要区域：  
1. 左上区域：X光管结构示意图。包含一个椭圆形真空管（标注"Vacuum"），左侧是阴极（"Cathode"，螺旋状线圈，标注"U_H"），右侧是倾斜的阳极（"Anode"，矩形块）。管内有五条红色箭头表示电子（"e⁻"）从阴极射向阳极，阳极处有红色波浪线表示X射线（标注"h·ν"）通过"Window"窗口射出。管外有电源符号标注"U"（"+"和"-"极）。  
2. 右上区域：X射线光谱图（标注"a)"）。横轴双刻度：上为"Energy / keV"（100-2），下为"λ / 10⁻¹⁰ m"（0.2-5.0）。纵轴无标签但显示强度分布。红色曲线显示连续谱，峰值在约50 keV，曲线上有垂直红色谱线标注：K_α₁, K_α₂, L_β₁, L_α₁, M_α₁等。图上方标注"Tungsten anode U = 100 kV"。  
3. 左下区域：X射线实验装置示意图。左侧是"X-ray tube"（圆柱形，带"W"符号），射出红色直线光束经两个"Collimators"（矩形狭缝）后入射到"Crystal"（倾斜矩形块），反射光束进入"Detector"（灰色矩形）。标注角度符号"θ"和"2θ"，以及"Matched filters"指向滤光片。  
4. 右下区域：多电压X射线光谱图。横轴"Wavelength λ / 10⁻¹⁰ m"（0.2-1.0），纵轴"Relative intensity"（0-12）。网格背景上有六条曲线，分别标注"20 kV"、"25 kV"、"30 kV"、"35 kV"、"40 kV"、"50 kV"，曲线从左到右强度先升后降，峰值随电压升高向短波长移动。  
页面底部有黑色文字"轫致辐射(bremsstrahlung，刹车辐射)"，其下方是两个公式（第一个行内公式，第二个行间公式），最底部是杜安和亨特的实验描述文字。整体布局为2×2网格，文字与图形穿插，关键术语和标题用红色突出。
