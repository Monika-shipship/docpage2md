# X射线与Compton效应-X射线的发现

X射线试鞋机

## Figure & Layout Description
- **整体布局**：页面采用纵向排版，分为三个主要区域：顶部标题栏、中部图像区、底部说明文字区。标题栏与图像区之间有一条细的深灰色水平分隔线。
- **标题栏**：位于页面最上方，包含黑色粗体文字“X射线与Compton效应-”和红色粗体文字“X射线的发现”，文字大小一致（约28pt），整体居中对齐。红色部分“X射线的发现”使用醒目的红色（#FF0000）以强调主题，与黑色部分形成强烈对比。
- **图像区**：占据页面主体，分为左右两个子区域：
  - **左侧子区域**：包含两张竖直排列的黑白历史照片，背景为浅灰色。上方照片中，一位穿西装的男士正操作一台大型设备，旁边站着一位持花束的女士和一位小女孩；照片顶部有黑色斜体大写字母“X-RAY”，底部有黑色大写字母“CUSTOMERS”。下方照片中，一位穿长裙的女士站在设备上，一位男士正在调整设备；照片顶部有黑色斜体大写字母“FITTING”，底部有黑色大写字母“EXPECT IT!”，其中“IT!”后有一个感叹号。
  - **右侧子区域**：包含多组图像和文字元素。顶部有黑色大写字母“3. X-RAY FITTING TEST”。其下有三部分：左侧是脚部X光图像，标注黑色文字“RIGHT WAY”；中间是两人操作设备的示意图，标注“X-RAY TEST”；右侧是另一脚部X光图像，标注“WRONG WAY”。中间区域下方有测试选项表：左侧列标题“LEFT”对应三个选项框（“GOOD”、“FAIR”、“POOR”），右侧列标题“RIGHT”对应三个空选项框。最下方是一个放大的脚部X光图像，清晰显示脚骨与鞋内结构，鞋底有网格状标记。
- **底部说明区**：位于页面最下方，居中显示蓝色粗体文字“X射线试鞋机”（颜色#0000FF），字体大小约18pt，与标题栏形成呼应。
- **颜色与层级**：整体以黑白灰为主色调，仅标题中“X射线的发现”和底部文字使用彩色（红/蓝）突出关键信息。图像区左侧照片背景为浅灰，右侧测试图背景为白色，形成视觉分区。文字层级分明：标题最大，图像标注次之，测试选项最小。
