# 原子的内层能级和特征X射线-X射线能谱

## X射线特征谱

当加速电压大于一定值时，在连续谱上出现了分立的线谱：

(1) 其波长与加速电压无关，只与靶材料有关，因此称为特征谱。

(2) 不同的靶材料，除了波长不同外，X射线的特征谱具有相似的结构。

## Figure & Layout Description

右侧包含一个坐标图，横轴为WAVELENGTH（angstoms），范围0到3.0；纵轴为X-RAY INTENSITY（relative units），范围0到6。图中绘制了5条黑色细线曲线，分别标注"5"、"10"、"15"、"20"、"25 kV"，表示不同加速电压下的X射线强度分布。在波长约0.5-1.0 angstroms区域出现两处尖锐的特征辐射峰，标注有"Kα"（强度峰值约5）和"Kβ"（强度峰值约3）。图中用黑色箭头分别指向"characteristic radiation"（特征辐射，尖峰部分）和"continuous radiation"（连续辐射，平滑曲线部分）。坐标网格线为浅灰色，特征峰用黑色实线标注，左下角有"SWL"标注指示短波长极限位置。整体采用黑白配色方案，文字标注使用无衬线字体。
