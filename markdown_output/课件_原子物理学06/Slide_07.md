# X射线与Compton效应-**X射线的发现**

## 晶体衍射

1912年，小布拉格提出一种更简便的晶体衍射方法。规则排列的原子形成布拉格平面，X射线从相邻平面散射，形成干涉。

$$n\lambda = 2d \sin\theta$$

布拉格条件(相干加强) $2d \sin \theta = n\lambda$, $n = 1, 2, 3...$
