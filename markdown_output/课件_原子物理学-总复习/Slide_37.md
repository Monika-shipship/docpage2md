# 电子自旋

## 电子的自旋磁矩

$$\mu_s = -\frac{e}{m_e} \mathbf{S} = -g_s \frac{\mu_B}{\hbar} \mathbf{S}$$

$$\mu_{sz} = -\frac{e}{m_e} S_z = -g_s m_s \mu_B$$

其中 $g_s = 2$，称作电子的自旋朗德(A. Landé) g因子，它可以表示为：

$$g_s = \frac{|\mu_{sz} / \mu_B|}{|S_z / \hbar|}$$

也称为电子自旋回磁比。

## 电子的轨道磁矩

$$\mu_l = -\frac{e}{2m_e} \mathbf{L} = -g_l \frac{\mu_B}{\hbar} \mathbf{L}$$

$$\mu_{lz} = -\frac{e}{2m_e} L_z = -g_l m_l \mu_B$$

$$g_l = \frac{|\mu_{lz} / \mu_B|}{|L_z / \hbar|}$$

$g_l$ 称作轨道g因子，$g_l = 1$。
