# X射线与Compton效应-Compton效应

## 光子的散射

### 能量守恒
$$h\nu_0 + m_0 c^2 = h\nu_S + \sqrt{p^2 c^2 + m_0^2 c^4}$$

### 动量守恒
$$\frac{h}{\lambda_0} = \frac{h}{\lambda_S} \cos\varphi + mv \cos\theta$$
$$\frac{h}{\lambda_S} \sin\varphi = mv \sin\theta$$

$$\lambda_S - \lambda_0 = \frac{h}{m_0 c}(1 - \cos\varphi)$$

$$\lambda_C = \frac{h}{m_0 c} \approx 0.00243 \text{nm} \quad \text{电子的Compton波长}$$
