# 轨道角动量

## 角动量空间取向量子化

$L = \sqrt{l(l+1)}\hbar$ \quad $L_z = m_l \hbar$

对于给定量子数 $l$，$m_l = 0, \pm 1, \pm 2, \cdots \pm l$

$\cos \theta = \frac{L_z}{|L|}$

$\cos \theta = \frac{0, \pm \hbar, \pm 2\hbar}{|L|}$

Vector Model for Orbital Angular Momentum

左侧示意图展示三维坐标系：x轴为紫色箭头（标注红色"x"），y轴为黄色箭头（标注红色"y"），z轴为红色箭头（标注红色"z"）。从原点出发有一个红色向量标注"$L$"，其z方向分量用蓝色向量标注"$L_z$"，并环绕z轴绘制蓝色椭圆表示$L_z$的旋转轨迹。角度$\theta$用红色弧线标注在$L$与z轴之间。坐标系底部有蓝色公式"$\cos \theta = \frac{L_z}{|L|}$"。
