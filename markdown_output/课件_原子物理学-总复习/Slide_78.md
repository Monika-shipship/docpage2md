## j-j耦合

**例：**  
对于多重态$(1/2, 3/2)$  
$j_1 = 1/2$, $j_2 = 3/2 \Rightarrow J = 2, 1$  

对于多重态$(1/2, 1/2)$  
$j_1 = 1/2$, $j_2 = 1/2 \Rightarrow J = 1, 0$  

形成两对能级：  
$(1/2, 1/2)_0$, $(1/2, 1/2)_1$  
和  
$(1/2, 3/2)_1$, $(1/2, 3/2)_2$。
