# 碱金属原子的能级与光谱

碱金属是元素周期表中的第一列元素：

锂(Li)、钠(Na)、钾(K)、铷(Rb)、铯(Cs)、钫(Fr)

原子实 + 结合松散的价电子

基态价电子处在$ns$态

**对于锂、钠、钾、铷、铯、钫，$n$ 分别等于2, 3, 4, 5, 6, 7。**
