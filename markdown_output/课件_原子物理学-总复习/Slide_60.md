# 外磁场中的原子—强磁场

$|-\mu \cdot B| >> |-\mu_s \cdot B_e|$

进一步考虑自旋—轨道耦合

$l \neq 0$  
$\Delta E_{ls} = \xi_{nl} m_l m_s$

$l = 0$  
$\Delta E_{ls} = 0$

$\xi_{nl}$是自旋—轨道耦合系数

| $m_l$ | $m_s$ | $\Delta E$         |
|--------|--------|---------------------|
| +1     | +1/2   | $2\mu_B B + \xi_{np}/2$ |
| 0      | +1/2   | $\mu_B B$         |
| (+/-)1 | (-/+)1/2 | $-\xi_{np}/2$    |
| 0      | -1/2   | $-\mu_B B$        |
| -1     | -1/2   | $-2\mu_B B + \xi_{np}/2$ |

| $m_l$ | $m_s$ | $\Delta E$ |
|--------|--------|-------------|
| 0      | +1/2   | $\mu_B B$ |
| 0      | -1/2   | $-\mu_B B$ |

## Figure & Layout Description

页面顶部是黑色粗体标题"外磁场中的原子—强磁场"，下方有一条深灰色水平分割线。标题下方左侧区域包含核心公式：红色显示的不等式$|-\mu \cdot B| >> |-\mu_s \cdot B_e|$，其右侧有一个橙色向上箭头指向该不等式。不等式下方是蓝色文字"进一步考虑自旋—轨道耦合"。

左侧主体区域分为两部分：
1. 上部：黑色文字"$l \neq 0$"及其对应的能量公式$\Delta E_{ls} = \xi_{nl} m_l m_s$
2. 下部：黑色文字"$l = 0$"及其对应的能量公式$\Delta E_{ls} = 0$
3. 底部：黑色文字说明"$\xi_{nl}$是自旋—轨道耦合系数"

右侧区域包含能级分裂示意图：
- 顶部标有"3p"的水平线向右延伸，连接到一组垂直排列的能级（共5层），用虚线表示原始能级，实线表示分裂后的能级
- 中部标有"3s"的水平线向右延伸，连接到一组垂直排列的能级（共2层）
- 能级之间有垂直向下的实线箭头表示跃迁，共6条跃迁线
- 每个能级右侧标注对应的$m_l$、$m_s$和$\Delta E$值，形成两组表格
- 底部有谱线分布示意图，从左到右依次标注"$\sigma$"、"$\pi$"、"$\sigma$"，下方有指向右侧的箭头标注"$\nu$"

整体布局为左文右图结构，文字部分使用黑色、红色、蓝色区分不同信息层级，图表部分使用实线、虚线和箭头表示能级结构和跃迁关系。
