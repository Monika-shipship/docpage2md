# 电子组态能级的修正

$$\hat{H}_1 = \sum_{i<j=1}^{N} \frac{e^2}{4\pi\varepsilon_0 r_{ij}} - \sum_{i=1}^{N} S(r_i)$$
$\hat{H}_1$ 大致与Z成比

$$\hat{H}_2 = \sum_{i=1}^{N} \xi(r_i) \mathbf{l}_i \cdot \mathbf{s}_i$$
$\hat{H}_2$ 大致与Z^4成比

这两项修正的重要性要看它们的相对大小。如果两项的大小相似，称为中间耦合，量子力学上处理比较困难。  
通常讨论两种极端的情形，对于基态原子和轻原子的低激发态，通常剩余静电势远大于自旋一轨道相互作用，这称为LS耦合。另一种极端的情形通常发生在重原子或高激发态，此时自旋一轨道相互作用远大于剩余静电势，这称为jj耦合。
