## Figure & Layout Description
页面顶部有黑色粗体主标题"泡利不相容原理和交换效应—两个电子的自旋波函数"，下方有一条深灰色水平分隔线。主体内容分为左右两栏：左栏为数学表达式和文字说明，右栏为三组自旋状态示意图。左栏顶部显示" $S = 1$ "黑色公式，其右侧是红色汉字"三重态"；下方依次排列三个自旋波函数表达式：第一行是$σ_{+}(1)σ_{+}(2)$，第二行是$\frac{1}{\sqrt{2}}[σ_{+}(1)σ_{-}(2) + σ_{-}(1)σ_{+}(2)]$，第三行是$σ_{-}(1)σ_{-}(2)$；左栏底部有黑色说明文字"两个电子的自旋取向看作是“平行”的"。右栏包含三行图示：第一行展示$M_s = +1$状态，含z轴坐标系，两个红色实线自旋矢量$\vec{s_1}$和$\vec{s_2}$均指向上方，标注$m_{s_1} = +\frac{1}{2}$和$m_{s_2} = +\frac{1}{2}$，底部标有$M_s = +1$；第二行展示$M_s = 0$状态，左侧子图(a)含z轴和两个红色自旋矢量（$\vec{s_1}$向上、$\vec{s_2}$向下），标注$m_{s_1} = +\frac{1}{2}$和$m_{s_2} = -\frac{1}{2}$；右侧子图显示交换后的矢量组合，标注$m_{s_2} = +\frac{1}{2}$和$m_{s_1} = -\frac{1}{2}$，中间有"+"符号连接，最终合成矢量$\vec{S}$指向右侧，标注$|\vec{S}| = \sqrt{2} \cdot \hbar$；第三行展示$M_s = -1$状态，z轴坐标系中两个红色自旋矢量均指向下方，标注$m_{s_1} = -\frac{1}{2}$和$m_{s_2} = -\frac{1}{2}$，底部标有$M_s = -1$。所有自旋矢量均用红色实线箭头表示，部分图示包含红色虚线椭圆表示自旋进动轨迹，坐标轴z用黑色实线箭头表示。
