# 泡利不相容原理和交换效应—交换效应

氦原子中的两个电子可以**分别处在**：1s, 2s, 2p, 3s, 3p, ....

电子组态：1s1s 或1s²；1s2s; ...

$$E^{(0)} = E_{n_1} + E_{n_2} = -\frac{Z^2}{2}m_e c^2 \alpha^2 \left(\frac{1}{n_1^2} + \frac{1}{n_2^2}\right)$$

$Z = 2$

**He原子能级**

原子中两个电子同时激发的概率很小，如果没有特殊说明，我们只讨论**一个电子激发的情形**。
