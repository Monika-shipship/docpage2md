## Figure & Layout Description

页面顶部有黑色粗体标题“He原子的光谱与能级”，下方配有一条深灰色水平分隔线。主体内容由三个水平排列的原子结构示意图组成，从左至右通过两个蓝色右向箭头连接，表示概念演进关系。

左侧图示：  
- 蓝色细线绘制的圆形轨道，轨道上有一个绿色小圆点标注红色文字“$-e, m_e$”  
- 轨道中心为红色实心圆，标注红色文字“$+Ze, M$”  
- 图形正下方有蓝色文字“氢原子(类氢离子)”

中间图示：  
- 浅蓝色填充的大圆形区域（表示电子云），内部中心为红色实心圆标注红色文字“$+Ze$”  
- 电子云内部有红色文字标注“$-(Z-1)e$”  
- 电子云右上角悬浮一个蓝色小圆点，标注红色文字“$-e$”  
- 图形正下方有蓝色文字“碱金属原子”

右侧图示：  
- 深蓝色细线绘制的圆形轨道，轨道上有两个蓝色小圆点均标注红色文字“$-e$”  
- 轨道中心为红色实心圆，标注红色文字“$+2e$”  
- 图形正下方有蓝色文字“氦原子”

所有电荷标注文字均为红色，原子类型标注文字均为蓝色。轨道线颜色从左到右依次为标准蓝、无轨道线（电子云）、深蓝色，电子标记点颜色从左到右为绿色、蓝色、蓝色。
