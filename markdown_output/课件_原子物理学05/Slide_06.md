# He原子的光谱与能级

## 全同粒子

所有内禀属性都相同的粒子称为全同粒子

例如所有的电子都具有相同的静止质量$0.511 \text{ MeV}/c^2$，相同的电荷$e$，相同的$\frac{1}{2}$自旋等等，我们不能通过实验观测到不同电子个体之间内禀属性的差异。

*Identical Robots*  
*Identical particles*
