# 泡利不相容原理和交换效应—交换效应

## (3) 激发态能级，$1s n l$

**空间波函数**  
$$u_{S,A}^{(0)}(\boldsymbol{r}_1,\boldsymbol{r}_2) = \frac{1}{\sqrt{2}}[u_{100}(\boldsymbol{r}_1)u_{nlm_l}(\boldsymbol{r}_2) \pm u_{nlm_l}(\boldsymbol{r}_1)u_{100}(\boldsymbol{r}_2)]$$

**讨论更一般的情况**  
$$u_{S,A}^{(0)}(\boldsymbol{r}_1,\boldsymbol{r}_2) = \frac{1}{\sqrt{2}}[u_{n_1 l_1 m_{l1}}(\boldsymbol{r}_1)u_{n_2 l_2 m_{l2}}(\boldsymbol{r}_2) \pm u_{n_2 l_2 m_{l2}}(\boldsymbol{r}_1)u_{n_1 l_1 m_{l1}}(\boldsymbol{r}_2)]$$

**能量**  
$$E^{(0)} = E_{n_1} + E_{n_2} = -\frac{Z^2}{2}m_e c^2 \alpha^2 \left( \frac{1}{n_1^2} + \frac{1}{n_2^2} \right)$$

**计算库仑斥力势能的平均值**  
$$\Delta E = \frac{e^2}{4\pi\varepsilon_0} \left\langle \frac{1}{r_{12}} \right\rangle$$  
$$\Delta E = K \pm J$$
