## 泡利不相容原理和交换效应—交换效应

双电子的空间波函数  
$$u^{(0)}(\mathbf{r}_1, \mathbf{r}_2) = u_{n_1 l_1 m_1}(\mathbf{r}_1) u_{n_2 l_2 m_2}(\mathbf{r}_2)$$

体系的总能量  
$$E^{(0)} = E_{n_1} + E_{n_2} = -\frac{Z^2}{2} m_e c^2 \alpha^2 \left( \frac{1}{n_1^2} + \frac{1}{n_2^2} \right)$$

或  
$$u^{(0)}(\mathbf{r}_1, \mathbf{r}_2) = u_{n_2 l_2 m_2}(\mathbf{r}_1) u_{n_1 l_1 m_1}(\mathbf{r}_2)$$

体系的总能量相同。 这种特殊的简并称为“交换简并”。

空间波函数必须是交换对称或反对称的，则合适的空间波函数为：  
$$u_{S,A}^{(0)}(\mathbf{r}_1, \mathbf{r}_2) = \frac{1}{\sqrt{2}} \left[ u_{n_1 l_1 m_1}(\mathbf{r}_1) u_{n_2 l_2 m_2}(\mathbf{r}_2) \pm u_{n_2 l_2 m_2}(\mathbf{r}_1) u_{n_1 l_1 m_1}(\mathbf{r}_2) \right]$$
