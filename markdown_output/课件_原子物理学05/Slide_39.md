## Figure & Layout Description

页面包含三个垂直排列的能级图，整体布局分为左、中、右三部分，每部分顶部有灰色背景条，内含红色文字"1s + 'free' electron continuum"。每个区域通过绿色竖线分隔：

1. **左侧区域**：
   - 顶部有蓝色点线表示的连续谱（"free" electron continuum）
   - 两条水平蓝色实线分别标注"1s2s"和"1s2p"能级
   - 底部有单独的蓝色实线标注"1s1s"
   - 区域底部标注能量表达式 $E_{1s} + E_n$

2. **中间区域**：
   - 顶部同样有蓝色点线连续谱
   - 红色实线标注"1s2s"和"1s2p"能级（位置较左侧上移）
   - 蓝色虚线表示的能级结构位于红色线下方
   - 单独的红色实线标注"1s1s"
   - 底部标注能量表达式 $E_{1s} + E_n + K$

3. **右侧区域**：
   - 顶部有蓝色点线连续谱
   - 以绿色竖线分为左右两部分：
     - 左半部分标注 $S=0$，红色实线"1s2s"位于"1s2p"上方
     - 右半部分标注 $S=1$，红色实线"1s2s"位于"1s2p"下方
   - 蓝色虚线在 $S=0$ 区域显示为双线结构，在 $S=1$ 区域显示为单线
   - 底部标注能量表达式 $E_{1s} + E_n + K \pm J$
   - 通过水平虚线连接中间区域与右侧区域的对应能级，显示能级分裂现象

所有能级标注均使用红色字体，能量表达式使用黑色数学符号，连续谱统一用蓝色点线表示。红色实线表示占据态能级，蓝色虚线表示未占据态或关联效应。
