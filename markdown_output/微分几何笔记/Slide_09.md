# Slide 9

合同变换的几何意义：平移，旋转，镜面反射的复合

$T(x) = xT + P$

$\det(T) = 1$ 平移+旋转 刚体运动  
$\det(T) = -1$ 一定包含镜面的 反向刚体运动

定义一个集合  
$\{ T: T \text{是合同变换} \}$ + 定义运算满足四定律

得群 Group，称作合同变换群

$T_1(x) = xT_1 + P_1$  
$T_2(x) = xT_2 + P_2$

$T_1 \circ T_2 = xT_2 T_1 + P_1$  
$\quad = (xT_2 + P_2)T_1 + P_1$  
$\quad = xT_2 T_1 + P_2 T_1 + P_1$

单位元 $I(x) = x = xI + 0$
