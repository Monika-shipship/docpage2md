# Slide 61

$$R^{\mu}_{\nu} - \frac{1}{2} g^{\mu}_{\nu} R = \frac{8\pi G}{c^4} T^{\mu}_{\nu}$$

令 $\nu = \mu$，$g^{\mu}_{\mu} = 4$.

$$R - 2R = \frac{8\pi G}{c^2} T$$
$$R = -\frac{8\pi G}{c^2} T$$

而 $G^{\mu}_{\nu} = R^{\mu}_{\nu} - \frac{1}{2} g^{\mu}_{\nu} R$，
$$\nabla_{\mu} (G^{\mu}_{\nu}) = \nabla_{\mu} \left( R^{\mu}_{\nu} - \frac{1}{2} g^{\mu}_{\nu} R \right) = 0$$
协变散度为 0.

故 $T^{\mu\nu}$ 的协变散度为 0.
$$\nabla_{\mu} T^{\mu\nu} = 0$$
