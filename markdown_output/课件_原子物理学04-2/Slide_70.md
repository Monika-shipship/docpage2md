## 原子束磁共振

a)  
左侧为能级分裂示意图：  
- 纵轴 $E$（能量），横轴 $B$（磁场强度）  
- 标注能级：$2^2P_{3/2}$（分裂为 $m_j = +3/2, +1/2, -1/2$）、$2^2S_{1/2}$、$2^2P_{1/2}$  
- 红色双向箭头标注 **Lamb-Shift**（$2^2S_{1/2}$ 与 $2^2P_{1/2}$ 能级差）  
- 两处 **HF** 标签指示超精细结构分裂方向  
- $m_j$ 量子数标注：$+3/2, +1/2, -1/2, +1/2, -3/2, +1/2, -1/2$  

b)  
右侧为频率-磁场关系图：  
- 纵轴 $v/\text{GHz}$（频率），横轴 $B/\text{T}$（磁场强度）  
- 三条跃迁曲线：  
  - $S_{1/2}(m_j = \frac{1}{2}) \to P_{3/2}(m_j = -\frac{1}{2})$（上部红色点线）  
  - $S_{1/2}(m_j = \frac{1}{2}) \to P_{1/2}(m_j = -\frac{1}{2})$（中部红色点线）  
  - $S_{1/2}(m_j = \frac{1}{2}) \to P_{1/2}(m_j = \frac{1}{2})$（下部红色点线）

## Figure & Layout Description
页面顶部为黑体中文标题"原子束磁共振"，下方有一条深蓝色水平分隔线。主体内容分为左右两部分：  
1. **左侧图(a)**：  
   - 坐标系纵轴向上标 $E$（能量），横轴向右标 $B$（磁场强度），原点标记为 0  
   - 三条主能级线：  
     - 最上方 $2^2P_{3/2}$ 分裂为三条斜线（对应 $m_j = +3/2, +1/2, -1/2$）  
     - 中间 $2^2S_{1/2}$ 与 $2^2P_{1/2}$ 交叉区域用红色双向箭头标注 "Lamb-Shift"  
     - $2^2S_{1/2}$ 和 $2^2P_{1/2}$ 分别分裂为两条斜线  
   - 两处 "HF" 标签用黑色箭头指示超精细分裂方向  
   - 右侧垂直排列 $m_j$ 量子数（$+3/2$ 至 $-1/2$）  
2. **右侧图(b)**：  
   - 坐标系纵轴 $v/\text{GHz}$（刻度 1-10），横轴 $B/\text{T}$（刻度 0-0.2）  
   - 三条红色数据点曲线，每条曲线末端有黑色箭头指向对应跃迁标注：  
     - 最上方曲线：$S_{1/2}(m_j = \frac{1}{2}) \to P_{3/2}(m_j = -\frac{1}{2})$  
     - 中间曲线：$S_{1/2}(m_j = \frac{1}{2}) \to P_{1/2}(m_j = -\frac{1}{2})$  
     - 最下方曲线：$S_{1/2}(m_j = \frac{1}{2}) \to P_{1/2}(m_j = \frac{1}{2})$  
3. **底部图注**：  
   - 黑色小字标注 "Fig. 5.37." 及英文说明，分(a)(b)两部分描述图表内容
