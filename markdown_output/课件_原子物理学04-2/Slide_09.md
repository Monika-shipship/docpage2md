# 氢原子能级的超精细结构

## 原子核的自旋运动与电子运动相互作用

电子的轨道运动和电子自旋运动都会在原子核处产生磁场，可以证明：原子核感受到的电子的磁场  
$B_e \propto$ 电子总角动量$J$

核磁矩 $\mu_I = g_I \mu_N I / \hbar$  
$\Delta E = A \mathbf{I} \cdot \mathbf{J}$ $A$ 称为超精细相互作用常数

引入原子体系的总角动量$\mathbf{F}$  
$\mathbf{F} = \mathbf{I} + \mathbf{J}$  
$\mathbf{F}^2 = F(F+1)\hbar^2$  
$F_z = m_F \hbar$

$F$是总角动量量子数，  
$F = (I+J), (I+J-1), \cdots, |I-J|$  
$M_F = -F, -F+1, \cdots, F-1, F$
