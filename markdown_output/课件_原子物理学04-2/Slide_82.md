# 原子磁力仪的发展-SERF

## 开展研究
- 2008年：开始研究
- 2012.05：原理验证 $80\text{fT}/\text{Hz}^{1/2}$ (40Hz)
- 2013.01：方案验证 $8\text{fT}/\text{Hz}^{1/2}$ (40Hz) (项目启动)
- 2016.06：第一代装置 $0.68\text{fT}/\text{Hz}^{1/2}$ (13Hz)
- 2019.06：第二代装置 $0.089\text{fT}/\text{Hz}^{1/2}$ (30-39Hz) (项目验收)
- 2025：目标：$0.015\text{fT}/\text{Hz}^{1/2}$ (20-100Hz)

## 开始研究
- 2008年：开始研究
- 2013年：磁场测量研究平台 $8\text{fT}/\text{Hz}^{1/2}$ (40Hz)
- 2014年：小型化实验研究装置 $20\text{fT}/\text{Hz}^{1/2}$ (10-48Hz)
- 2016年：高精度磁强计样机 $8.6\text{fT}/\text{Hz}^{1/2}$ (10-48Hz)
- 2022年：体积≤20cm×10cm×10cm 高精度计量原子磁强计 $2\text{fT}/\text{Hz}^{1/2}$ (20-100Hz)
- 2026年：体积≤15cm×10cm×10cm 高精度计量原子磁强计 $1\text{fT}/\text{Hz}^{1/2}$ (20-100Hz) 本项目

## Figure & Layout Description
PPT页面采用三段式垂直布局结构。顶部为标题区域：黑色粗体汉字"原子磁力仪的发展-SERF"（约36pt）居中显示，下方有深灰色1.5pt水平分割线。中间主体区域分为左右两栏：左侧占60%宽度，展示双时间线发展图；右侧占40%宽度，为光学系统示意图。左侧时间线包含两条平行红色水平时间轴，上方标注"开展研究"（红色竖排文字），下方标注"开始研究"（红色竖排文字）。每条时间轴上分布6个时间节点，每个节点包含：年份标注（黑色小字）、对应阶段名称（蓝色/红色文字）、性能指标（黑色文字含单位下标）、以及小型设备实物照片（5-6张）。时间轴文字使用标准黑体，性能指标中单位如$\text{fT}/\text{Hz}^{1/2}$的下标格式精确呈现。右侧示意图采用黑白线条绘制，红色实线表示光路路径，黑色线条勾勒组件轮廓；组件标注文字为黑色无衬线字体，通过引线指向对应部件；右上角有三维坐标系（x,y,z轴）标注。底部区域为全宽建筑照片：展示现代科研建筑群，中央为金色穹顶主建筑，两侧对称分布红砖结构建筑，周围环绕绿化和水系；照片上方叠加深蓝色横幅，内有白色机构名称文字。整体色彩方案以白底为主，红色用于时间轴和光路强调，关键性能数据用蓝色突出显示，形成清晰的视觉层次。
