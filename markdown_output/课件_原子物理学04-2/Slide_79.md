# 光泵磁力仪（塞曼效应）

图 1.11 光泵磁力仪外观与实物图，从左到右依次为  
氦（He）光泵磁力仪、铷（Rb）光泵磁力仪、铯（Cs）光泵磁力仪

$\Delta m_F = 0, \pm 1$  
$\Delta m_F = +1$（左旋 $\sigma^+$），$\Delta m_F = -1$（右旋 $\sigma^-$）

法国物理学家卡斯特勒（A. Kastler）提出了一种光泵浦（optical Pumping）方法，即利用光对原子布居数进行转移。以圆偏振光为例，因左旋或右旋圆偏振光具有角动量$+h$或$-h$，根据角动量守恒，当该圆偏振光入射原子并作用时，原子只能吸收$+h$或$-h$的角动量

## Figure & Layout Description
页面顶部为黑色粗体标题"光泵磁力仪（塞曼效应）"，下方有深灰色水平分割线。主体区域包含三张横向排列的黑白仪器照片：左侧为带线缆的矩形设备（氦光泵磁力仪），中间为长筒形带支架的精密仪器（铷光泵磁力仪），右侧为手持式圆柱形设备（铯光泵磁力仪）。照片下方是图注说明，采用标准宋体字。中部偏左区域显示两个量子力学选择定则公式，使用标准数学符号排版。右侧为原子能级示意图，包含四条水平能级线（标注$E_1$、$E_1 + \Delta E_1$、$E_2$、$E_2 + \Delta E_2$）及两条垂直跃迁箭头（标注$h\nu_0$和$h\nu$）。底部左侧为卡斯特勒光泵浦理论的说明文字，整体采用黑底白字的学术PPT标准配色方案，文字与图形区域划分清晰，重点公式和能级图通过位置对比形成视觉焦点。
