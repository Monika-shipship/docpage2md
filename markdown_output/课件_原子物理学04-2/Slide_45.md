# 外磁场中的原子—弱磁场

平行于$J$的分量

$$\mu_j = \frac{\mu \cdot J}{J^2} J$$

其中

$$\mu \cdot J = -\frac{\mu_B}{\hbar}(L \cdot J + 2S \cdot J)$$

又

$$J = L + S$$
↓
$$S = J - L$$
↓
$$S^2 = (J - L)^2 = J^2 + L^2 - 2L \cdot J$$
↓
$$L \cdot J = \frac{1}{2}(J^2 + L^2 - S^2)$$

同样有

$$S \cdot J = \frac{1}{2}(J^2 + S^2 - L^2)$$

## Figure & Layout Description

页面右侧为三维矢量示意图，左侧为公式推导区域。示意图包含以下元素：
1. **坐标系**：竖直向上的z轴标注"z, $\vec{B}$"，代表外磁场方向
2. **矢量表示**：
   - 红色粗箭头$\vec{J}$斜向右上方，标注在椭圆轨迹末端
   - 红色箭头$\vec{L}$比$\vec{J}$更倾斜
   - 黑色箭头$\vec{S}$连接$\vec{L}$与$\vec{J}$末端
   - $J_z$用水平虚线表示$\vec{J}$在z轴上的投影
3. **轨迹形态**：
   - $\vec{J}$的进动轨迹为水平椭圆（实线）
   - $\vec{L}$和$\vec{S}$的进动轨迹为锥形（$\vec{L}$为红色锥面，$\vec{S}$为黑色锥面）
4. **磁矩分量**：
   - 蓝色箭头$\mu_j$平行于$\vec{J}$
   - 红色总磁矩$\mu$包含垂直分量（虚线表示）
   - $\mu_l$和$\mu_s$分别标注在对应锥面上
5. **颜色编码**：$\vec{J}$和$\vec{L}$用红色表示，$\vec{S}$用黑色，磁矩分量$\mu_j$用蓝色，总磁矩$\mu$用红色，坐标轴为黑色
