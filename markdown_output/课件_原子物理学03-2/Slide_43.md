# 跃迁和选择定则-电偶极跃迁

$$c_i^* c_f u_i^* u_f e^{i(E_i - E_f)t/\hbar} + c_i c_f^* u_i u_f^* e^{-i(E_i - E_f)t/\hbar}$$

$\psi_f \rightarrow \psi_i$  
$\psi_i \rightarrow \psi_f$

- 受激辐射  
- 吸收  
- 自发辐射

## Figure & Layout Description
页面顶部为黑色粗体标题"跃迁和选择定则-电偶极跃迁"，下方有一条深灰色水平分隔线。标题下方是居中的电偶极跃迁矩阵元公式，公式包含两个复共轭项，其中上标*表示复共轭，指数项含$\hbar$符号。公式下方有两组跃迁标注：左侧标注"$\psi_f \rightarrow \psi_i$"上方有橙色向上箭头，右侧标注"$\psi_i \rightarrow \psi_f$"上方有橙色向上箭头。

页面中部并列展示三个能级跃迁示意图：
1. 左侧"受激辐射"图：包含上下两条水平线（上标$E_i, u_i$，下标$E_f, u_f$），两条红色波浪线从右侧指向向下黑色箭头，下方标注蓝色"受激辐射"文字
2. 中间"吸收"图：结构类似，单条红色波浪线指向向上黑色箭头，下方标注蓝色"吸收"文字
3. 下方"自发辐射"图：单条红色波浪线从向下黑色箭头右侧射出，下方标注蓝色"自发辐射"文字，再下方有黑色小字"量子电动力学才能解释"

所有能级线为黑色实线，跃迁箭头为黑色，辐射波浪线为红色，说明文字使用蓝色（主标题）和黑色（补充说明）。三个示意图呈2×2网格布局（右下角为空），整体视觉层次分明，重点突出辐射过程的差异。
