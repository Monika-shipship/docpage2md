# 跃迁和选择定则-Einstein辐射唯象理论

## Planck黑体辐射公式

$$I(\omega) = \frac{\hbar \omega^3}{\pi^2 c^3} \frac{1}{\exp\left(\frac{\hbar \omega}{k_B T}\right) - 1} \quad \leftrightarrow \quad I(\omega) = \frac{A_{fi}}{B_{if}} \frac{1}{\exp\left(\frac{\hbar \omega}{k_B T}\right) - \frac{B_{fi}}{B_{if}}}$$

相比较，得 $B_{fi} = B_{if}$

$$A_{fi} = \frac{\hbar \omega^3}{\pi^2 c^3} B_{fi}$$

$$B_{fi} = B_{if} = \lambda_{fi} = \frac{\omega^3}{6 \varepsilon_0 \hbar c^3} |\langle \mathbf{p} \rangle|^2 = \frac{\omega^3}{6 \varepsilon_0 \hbar c^3} \left| \int u_f^*(-er) u_i d\tau \right|^2$$
