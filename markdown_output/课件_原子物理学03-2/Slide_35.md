# Slide 35

# 单电子(H)原子-H原子中电子的概率分布

$|u_{nlm}(r, \theta, \varphi)|^2$

## Figure & Layout Description
页面顶部为黑色粗体标题"单电子(H)原子-H原子中电子的概率分布"，标题下方有一条细长的深灰色水平分隔线。主体区域分为左右两部分：左侧和中间占据约70%宽度，展示18个概率分布图像；右侧30%区域显示数学公式。

图像区域采用非规则网格布局，共5行：
- 第1行：2个正方形图像，水平排列。左图标签"(2,0,0)"，右图标签"(3,0,0)"。图像呈现同心圆环结构，中心为深蓝色实心圆，向外渐变为浅蓝色、绿色直至白色背景，表示概率密度从中心向外递减。
- 第2行：3个正方形图像。左图标签"(2,1,0)"，显示沿垂直轴对称的哑铃形（双瓣），中心深蓝色，边缘渐变；中图标签"(3,1,0)"，哑铃形外有额外浅色环；右图标签"(3,1,1)"，双瓣沿水平轴分布，中心有浅色节点区域。
- 第3行：4个正方形图像。左图标签"(2,1,1)"，双瓣沿对角线分布；中左图标签"(3,2,0)"，三瓣花形（沿z轴）；中右图标签"(3,2,1)"，四瓣十字形；右图标签"(3,2,2)"，双环哑铃形。
- 第4行：5个正方形图像。左图标签"(4,0,0)"，四层同心圆；其余依次为"(4,1,0)"（三层哑铃）、"(4,1,1)"（双环水平哑铃）、"(4,2,0)"（复杂花瓣形）、"(4,2,1)"（四瓣星形）。
- 第5行：5个正方形图像。左图标签"(4,2,2)"，双环水平双瓣；中左图"(4,3,0)"，七瓣花形；中图"(4,3,1)"，六瓣星形；中右图"(4,3,2)"，四瓣对称形；右图"(4,3,3)"，双瓣沿对角线。

所有图像均采用蓝-绿-白渐变色谱：深蓝色（#00008B）表示最高概率密度区域，浅蓝色（#4169E1）为中等密度，绿色（#32CD32）为低密度，白色背景表示概率密度趋近零。每个图像下方居中位置标注黑色小号字体的量子数标签$(n,l,m)$，格式为括号包裹的三元组。右侧公式区域以大号黑色字体显示$|u_{nlm}(r, \theta, \varphi)|^2$，其中$u$为斜体，下标$nlm$清晰，括号内变量$r,\theta,\varphi$为斜体。
