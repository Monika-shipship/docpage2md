# 薛定谔方程的物理意义---从经典力学到量子力学

- 经典力学能量关系：
  $E = \frac{1}{2}mv^2 + V = \frac{p^2}{2m} + V$

- 定态薛定谔方程
  $$\left(-\frac{\hbar^2}{2m}\nabla^2 + V(\mathbf{r})\right)u(\mathbf{r}) = Eu(\mathbf{r})$$

- 波函数（态矢量）
  $$\psi(\mathbf{r},t) = \psi_0 \exp\frac{i}{\hbar}(\mathbf{p} \cdot \mathbf{r} - Et)$$
  
  分别对时间和空间求导

- 经典力学：力学量(如位置、速度、动量、能量等)
- 量子力学："态矢量 (State Vector)"（波函数）
- 波函数不是一个具体的物理量，但它包含了所有经典力学量的概率信息
- 位置（Born）：
  $|\psi(x)|^2$
  
  单个电子在x点的概率分布

- 其他的物理量呢？
