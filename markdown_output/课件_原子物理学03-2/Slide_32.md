# 单电子(H)原子-H原子中电子的概率分布

## Figure & Layout Description
该PPT页面包含6个子图，以2行3列网格布局排列，所有子图横轴均为归一化半径 $r/a_0$。  
**第一行（径向波函数 $R_{nl}(r)$）**：  
- **左图**：纵轴 $R_{nl}(r)$ 范围 [0.0, 2.0]，横轴 $r/a_0$ 范围 [0, 12]。单条衰减曲线标注 $R_{10}$，从 (0, ~2.0) 快速下降至 (12, ~0.0)。  
- **中图**：纵轴 $R_{nl}(r)$ 范围 [-0.2, 1.0]，横轴 $r/a_0$ 范围 [0, 12]。两条曲线：$R_{20}$（初始值 0.6，单峰后衰减）和 $R_{21}$（从 0 开始，先负后正，单节点）。  
- **右图**：纵轴 $R_{nl}(r)$ 范围 [-0.1, 0.4]，横轴 $r/a_0$ 范围 [0, 24]。三条曲线：$R_{30}$（单峰衰减）、$R_{31}$（双峰，含1节点）、$R_{32}$（三峰，含2节点）。  

**第二行（球壳概率密度 $r^2 R_{nl}^2(r)$）**：  
- **左图**：纵轴 $r^2 R_{nl}^2(r)$ 范围 [0.0, 0.8]，横轴 $r/a_0$ 范围 [0, 12]。单峰曲线标注 "1s"，峰值位于 $r/a_0 \approx 1$。  
- **中图**：纵轴 $r^2 R_{nl}^2(r)$ 范围 [0.00, 0.20]，横轴 $r/a_0$ 范围 [0, 16]。两条曲线："2s"（双峰，内峰较小）和 "2p"（单峰，峰值更高），"2p" 峰值位于 $r/a_0 \approx 5$。  
- **右图**：纵轴 $r^2 R_{nl}^2(r)$ 范围 [0.00, 0.12]，横轴 $r/a_0$ 范围 [0, 24]。三条曲线："3s"（绿色，三峰）、"3p"（红色，双峰）、"3d"（黑色，单峰），"3d" 峰值位于 $r/a_0 \approx 9$。  

所有子图采用黑色线条绘制，第二行中图和右图通过颜色区分轨道类型（2p/3p为红色，3s为绿色，3d为黑色），其余为单色。坐标轴刻度清晰，标注单位为 $a_0$（玻尔半径）。
