## 多电子原子—中心力场近似与电子组态

单电子的薛定谔方程

$$\left(-\frac{\hbar^2}{2m_e}\nabla_i^2 + V(r_i)\right)u(r_i) = E_i u(r_i), \ i=1,2,3,...,N$$

本征函数 $u(\mathbf{r}_i) = R_{n_i l_i}(r_i) Y_{l_i m_{l_i}}(\theta, \phi)$

取决于$V(r_i)$的具体形式  球谐函数

考虑单电子的自旋波函数

单电子的总波函数

$$\psi(q_i) = \psi(\mathbf{r}_i, s_i) = R_{n_i l_i}(r_i) Y_{l_i m_{l_i}}(\theta, \phi) \sigma_{m_{s_i}}$$

$$s = 1/2, \ m_s = \pm 1/2$$
