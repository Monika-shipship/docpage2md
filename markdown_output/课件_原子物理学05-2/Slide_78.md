# 多电子原子-电子组态能级的修正-- LS与jj耦合比较

相同原子的能级系列中，随激发电子$n$增大，能级趋向于$jj$耦合。这是因为当两个电子中的一个电子趋向高激发态时，两个电子之间的静电相互作用迅速减小，静电非中心力也迅速减小。

页面采用标准学术PPT布局，整体为黑白配色方案。顶部是加粗的一级标题"多电子原子-电子组态能级的修正-- LS与jj耦合比较"，标题下方有一条深灰色水平分隔线（约2像素粗）。正文段落位于标题下方，采用常规字号的黑色宋体文字。

左侧LS耦合区域：
- 顶部标注"L-S coupling"英文说明

中间区域：
- 顶部标注"Si 3pns"作为中心标题
- 底部水平排列四个组态标签："3p4s"、"3p5s"、"3p6s"、"3p7s"，表示主量子数n从4到7的激发态

右侧jj耦合区域：
- 顶部标注"jj coupling"英文说明

图表中的连接线采用点划线样式，表示能级随n增大的演化路径。左侧LS耦合区域的能级分裂明显，右侧jj耦合区域的能级结构更简单，直观展示了从LS耦合向jj耦合的转变趋势。所有文字和符号均为黑色，无背景色，图表线条粗细一致。
