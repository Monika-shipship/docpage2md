# 多电子原子—中心力场近似与电子组态

通常 $\hat{H}_0 \gg \hat{H}_1$

$$\hat{H}_1 = \sum_{i<j=1}^{N} \frac{e^2}{4\pi\epsilon_0 r_{ij}} - \sum_{i=1}^{N} S(r_i)$$

即中心力场近似

薛定谔方程为

$$\hat{H}_0 u^{(0)}(\mathbf{r}_1, \mathbf{r}_2, \cdots, \mathbf{r}_N) = E^{(0)} u^{(0)}(\mathbf{r}_1, \mathbf{r}_2, \cdots, \mathbf{r}_N)$$

分离变量

$$u^{(0)}(\mathbf{r}_1, \mathbf{r}_2, \cdots, \mathbf{r}_N) = \prod_{i=1}^{N} u(\mathbf{r}_i)$$

分解为 $N$ 个单电子的薛定谔方程

$$\left( -\frac{\hbar^2}{2m_e} \nabla_i^2 + V(r_i) \right) u(\mathbf{r}_i) = E_i u(\mathbf{r}_i), \, i=1,2,3,\ldots,N$$

## Figure & Layout Description

页面右侧有一个原子结构示意图。中心是一个棕色圆形原子核，标注 "+Ze"，被浅绿色半透明圆形区域包围，代表中心力场作用范围。浅绿色区域内标注红色文字 "$S(r_i)$"。原子核周围分布着 $N$ 个电子：一个红色点标记为 "$i$"，其余蓝色点表示其他电子。从原子核出发有多个黑色向量箭头指向各电子，分别标注 $\vec{r}_1$、$\vec{r}_2$、$\vec{r}_i$、$\vec{r}_j$。红色点 "$i$" 与另一个蓝色点之间有红色向量箭头，标注 $\vec{r}_{ij}$。页面左侧公式中 $\frac{e^2}{4\pi\epsilon_0 r_{ij}}$ 项被红色圆圈划掉，表示该库仑相互作用项在中心力场近似中被处理。关键术语 "即中心力场近似" 使用蓝色字体突出显示，整体布局为左文右图结构，标题使用黑色粗体字并带有深蓝色下划线。
