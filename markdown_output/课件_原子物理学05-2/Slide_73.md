# 电子组态能级的修正-- jj耦合

中心力场近似下的电子组态能级的**简并**部分撤除，能级按照价电子$j_i$的不同组合产生分裂，其中最小的$j_i$组合的能级最低。

对于$v$个电子的组态，通常用$(j_1, j_2, \dots, j_v)$表示在$jj$耦合下的原子多重态(或谱项)。

例：$nsn'p$电子组态能级在$jj$耦合下的劈裂

对于$ns$电子，  
$l_1 = 0$，$s_1 = 1/2$，$j_1 = 1/2$

对于$n'p$电子，  
$l_2 = 1$，$s_2 = 1/2$，$j_2 = 3/2, 1/2$

分裂为2个多重态能级  
$\left( \frac{1}{2}, \frac{1}{2} \right)$ $\left( \frac{1}{2}, \frac{3}{2} \right)$

$H_0$ $H_0+H_2$
