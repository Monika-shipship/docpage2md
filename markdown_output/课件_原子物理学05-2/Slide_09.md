# 多电子原子—中心力场近似与电子组态

## 单电子状态

$$\psi(r, s) = R_{nl}(r) Y_{lm_l}(\theta, \phi) \sigma_{m_s}$$

四个量子数 $(n, l, m_l, m_s)$ 描述

## 单电子能量

$$E_{nl} = -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z_{nl}^{*2}}{n^2}$$

原子中的 $N$ 个电子可以分别处在：1s, 2s, 2p, 3s, 3p, ...

## 电子组态：

比如两电子组态 (He) $1s1s$ 或 $1s^2$；$1s2s$；...

三电子组态 (Li) $1s^22s$；$1s^22p$；$1s^23s$；...

$N$ 个电子组态 $\quad n_1 l_1, n_2 l_2, \cdots n_N l_N$
