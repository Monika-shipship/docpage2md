# Slide 64

## 电子组态能级的修正-LS耦合

Slater图解法求$np^2$电子组态的谱项

$$L=1, S=1; \, ^3P$$
$$L=0, S=0; \, ^1S$$

## Figure & Layout Description

图片包含三个水平排列的浅蓝色圆角矩形（背景色为#B0D0E0），每个矩形内构建二维坐标系：垂直轴标注$M_L$（黑色向上箭头），水平轴标注$M_S$（黑色向右箭头）。坐标系中数字以白色圆形背景配黑色字体显示。

1. **左侧矩形**：
   - 坐标系中心位置标有数字"2"，周围8个对称位置（上下左右及四个斜向）均标有数字"1"
   - 矩形右侧有一个深蓝色右向箭头（#00008B），指向中间矩形

2. **中间矩形**：
   - 坐标系中心位置标有数字"1"，周围8个对称位置均标有数字"1"
   - 矩形下方标注文字：$L=1, S=1; \, ^3P$
   - 矩形右侧有一个蓝色"+"符号（#0000FF），连接右侧矩形

3. **右侧矩形**：
   - 坐标系仅中心位置标有数字"1"，其余位置无数字
   - 矩形下方标注文字：$L=0, S=0; \, ^1S$

三个矩形通过箭头和加号形成"总态分布 → 允许态分解"的逻辑流程，整体布局呈现从左到右的推导关系。
