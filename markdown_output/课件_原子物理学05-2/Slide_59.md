## Figure & Layout Description

页面顶部为黑色粗体标题"电子组态能级的修正-LS耦合"，下方有一条深灰色水平分割线。标题下方是黑色常规字体说明文字"等效电子组态$np^2$，则受到 Pauli原理的限制。"。再下方是居中对齐的二级标题"两个$p$电子可能形成的原子态"。

主体部分是一个5行4列的表格，具有浅橙色边框和浅米色背景。表格左上角单元格被对角线分割，左下部分标注$M_L$，右上部分标注$M_S$。行标题为$M_L$取值：2, 1, 0, -1, -2；列标题为$M_S$取值：1, 0, -1。

表格内容包含多种符号组合：
- 使用方括号[]和圆括号()区分不同类型的态
- 符号包含数字与上标+/-组合（如$1^+$, $-1^-$）
- 有9个单元格被红色斜线划掉，具体位置为：
  * $M_L=2$行的$M_S=1$和$M_S=-1$列
  * $M_L=0$行的$M_S=1$和$M_S=-1$列
  * $M_L=-2$行的$M_S=1$和$M_S=-1$列
  * $M_L=0$行$M_S=1$列中$(0^+0^+)$和$(1^+-1^+)$部分
  * $M_L=0$行$M_S=-1$列中$(0^-0^-)$和$(1^--1^-)$部分
- 表格中部分符号使用红色字体标注（如$M_L=1$行$M_S=0$列中的$0^-1^+$）

所有文字均为黑色，除红色标注和划线外，表格整体采用简洁的学术风格排版。
