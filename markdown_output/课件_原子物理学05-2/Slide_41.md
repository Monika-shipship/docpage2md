# 电子组态能级的修正

(1) 剩余静电势

$$\hat{H}_1 = \sum_{i<j=1}^{N} \frac{e^2}{4\pi\varepsilon_0 r_{ij}} - \sum_{i=1}^{N} S(r_i)$$

由于 $$\sum_{m_l=-l}^{+l} Y_{lm_l}^* (\theta, \phi) Y_{lm_l} (\theta, \phi) = \frac{2l+1}{4\pi}$$

满支壳层电子组态的电荷分布是球对称的。

只要计及最外未满支壳层上的价电子即可。

$\hat{H}_1$ 大致与Z成比
