## 电子组态能级的修正-LS耦合

等效电子组态 $(nl)^v$ 和 $(nl)^{Y-v}$ $\quad$ $Y = 2(2l+1)$ 满支壳层电子数

对于满支壳层电子组态 $(nl)^Y$ $\quad$ $L = S = 0$

$$M_L = \sum_{i=1}^{Y} m_{l_i} = 0 \quad M_S = \sum_{i=1}^{Y} m_{s_i} = 0$$

有  
$$(M_L)_v + (M_L)_{Y-v} = (M_L)_Y = 0$$

→ $(M_L)_v = -(M_L)_{Y-v}$

有  
$$(L)_v = (L)_{Y-v}$$

同理  
$$(S)_v = (S)_{Y-v}$$

互补电子组态 $(nl)^v$ 和 $(nl)^{Y-v}$ 具有相同的原子态谱项。
