# 电子组态能级

中心力场近似下，多电子原子体系总能量

$$E^{(0)} = \sum_{i=1}^{N} E_{n_i l_i}$$

由电子组态决定 $n_1 l_1, n_2 l_2, \cdots n_N l_N$

电子组态能级：基态  
单电子激发态

$E_{1s} + E_{n,l}$

He原子的电子组态能级
