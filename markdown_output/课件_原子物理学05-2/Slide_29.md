# 多电子原子-元素周期律

## 第六周期

从铯(Cs: Cesium, $Z = 55$)到氡(Rn: Radon, $Z = 86$)共有32个元素，依次填充 $6s$, $4f$, $5d$, $6p$ 支壳层。

从铈(Ce: Cerium, $Z = 58$)起到镥(Lu: Lutetium, $Z = 71$) 新增加的电子不断填充 $4f$ 支壳层，这些元素形成**镧系元素**(包括镧，Lanthanum)。

填充 $5d$ 壳层的元素(从 $Z = 72$ 到 $Z = 80$)称为第三组过渡元素。
