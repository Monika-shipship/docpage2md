## 电子组态能级的修正-LS耦合

$J$ 满足角动量量子化关系

$$J^2 = J(J+1)\hbar^2$$

$$J_z = M_J \hbar$$

其中，$J$ 是总轨道角动量量子数。

$$J = L+S, L+S-1, \dots, |L-S|$$

$M_J$ 是相应的磁量子子数

$$M_J = J, J-1, \dots, -J$$

$J$ 和 $M_J$ 是好量子数。

## Figure & Layout Description

右侧包含一个三维矢量示意图：垂直向上的黑色实线箭头表示Z轴，其顶端标注"Z"。红色虚线绘制的水平椭圆环绕Z轴，椭圆中心引出一条红色虚线指向右侧标注为"J"的红色实线向量，该向量末端标注"J_z"。黑色实线锥体表示轨道角动量$L$的进动轨迹，锥体表面有螺旋线纹理；黑色实线向量$S$从原点指向锥体内，与$L$向量共同构成矢量合成关系。所有矢量均从坐标原点出发，红色向量$J$为$L$与$S$的矢量和。图中文字标注均为黑色，虚线椭圆和$J$向量使用红色突出显示，其余线条为黑色。
