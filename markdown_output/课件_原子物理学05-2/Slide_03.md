# 多电子原子—中心力场近似与电子组态

## **空间波函数**

满足薛定谔方程
$$\hat{H}u(\vec{r}_1, \vec{r}_2, \cdots, \vec{r}_N) = Eu(\vec{r}_1, \vec{r}_2, \cdots, \vec{r}_N)$$

哈密顿量
$$\hat{H} = \sum_{i=1}^{N} \left( -\frac{\hbar^2}{2m_i} \nabla_i^2 - \frac{Ze^2}{4\pi\varepsilon_0 r_i} \right) + \sum_{i<j=1}^{N} \frac{e^2}{4\pi\varepsilon_0 r_{ij}}$$

$\vec{r}_i$ 是第 $i$ 个电子相对核的坐标
$$r_{ij} = |\vec{r}_i - \vec{r}_j|$$
第二项求和遍及所有两两电子之间

He原子的结果表明：**直接忽略 $\sum_{i<j=1}^{N} \frac{e^2}{4\pi\varepsilon_0 r_{ij}}$ 太粗略。**

## Figure & Layout Description

页面采用左右分栏布局：左侧为理论公式区域，右侧为原子结构示意图。  
- **左侧区域**：  
  - 顶部是黑色粗体主标题"多电子原子—中心力场近似与电子组态"，下方有深灰色水平分割线。  
  - 红色加粗文字"空间波函数"作为二级标题，其下依次排列薛定谔方程、哈密顿量公式及说明文字。  
  - 所有公式均使用标准物理符号：$\hbar$（约化普朗克常数）、$\varepsilon_0$（真空介电常数）、$\nabla_i^2$（拉普拉斯算符）等，求和符号$\sum$的上下限清晰标注。  
  - 底部有黄色背景高亮框，内含黑色文字强调He原子计算结果。  

- **右侧区域**：  
  - 中心位置为棕色圆形原子核，标注"$+Ze$"。  
  - 原子核周围分布4个蓝色小圆点代表电子（整体标注"$N e^-$"），每个电子通过黑色箭头向量连接至原子核，向量分别标注$\vec{r}_1$、$\vec{r}_2$、$\vec{r}_i$、$\vec{r}_j$。  
  - 两个电子之间用红色箭头表示$\vec{r}_{ij}$，箭头从第$i$个电子指向第$j$个电子，清晰展示电子间相对位置。  

- **视觉特征**：  
  - 文字颜色：主标题黑色，二级标题红色，关键结论黄色高亮  
  - 图形元素：原子核（棕色实心圆）、电子（蓝色实心圆）、向量（黑色/红色箭头）  
  - 层级关系：原子核作为原点，所有向量均以核为中心向外辐射，形成明确的空间坐标系
