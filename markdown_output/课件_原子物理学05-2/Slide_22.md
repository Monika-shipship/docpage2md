# 多电子原子-元素周期律

## 第一周期

**H (Hydrogen) 氢**  
基态电子组态 **1s**  
电离能 $I.P. = |E_{1s}| = \left| -\frac{1}{2} m_e c^2 \alpha^2 \frac{1}{1^2} \right| = 13.6 \, \text{eV}$

**He (Helium) 氦**  
基态电子组态 **1s²**  
电离能 $I.P. = |E_{1s}| = \left| -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z_{1s}^{*2}}{1^2} \right| = Z_{1s}^{*2} \times 13.6 \, \text{eV}$  
$1 < Z_{1s}^* < 2$  
$13.6 \, \text{eV} < I.P. < 54.4 \, \text{eV}$  
实验值: $24.59 \, \text{eV}$

底部电子排布示意图：  
- H: $1^2 S_{1/2}$  
- He: $1^1 S_0$  
- 第二周期元素（Li至Ne）的K/L层s/p轨道电子填充状态（含自旋方向箭头）
