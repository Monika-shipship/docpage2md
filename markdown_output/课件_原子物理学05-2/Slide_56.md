## Figure & Layout Description

页面顶部是黑色粗体标题"电子组态能级的修正-LS耦合"，下方有一条深灰色水平分隔线。标题下方并列三个公式：前两个公式$M_L = \sum m_l$和$M_S = \sum m_s$带有淡红色背景高亮，第三个公式$(m_{l_1}^{m_{s1}} m_{l_2}^{m_{s2}})$带有淡绿色背景高亮。公式下方是居中显示的子标题"两个p电子可能形成的原子态"。主体内容是一个5行3列的表格，表格具有浅橙色背景和橙色边框。表格左上角单元格用斜线分隔标注"M_L"（行标题）和"M_S"（列标题）。行标题从上到下依次为2、1、0、-1、-2；列标题从左到右依次为1、0、-1。表格内单元格包含多种符号组合，其中上标"+"和"-"使用Unicode上标字符（⁺/⁻），部分条目用方括号[]和圆括号()区分不同组合类型，某些单元格内容通过换行符分隔多组数据。
