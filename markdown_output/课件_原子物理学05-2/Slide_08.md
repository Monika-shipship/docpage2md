# 多电子原子—中心力场近似与电子组态

## 单电子的薛定谔方程

$$\left( -\frac{\hbar^2}{2m_e} \nabla_i^2 + V(r_i) \right) u(r_i) = E_i u(r_i), \, N=1,2,3,\ldots$$

单电子的能量$E_{nl}$不仅与主量子数$n$有关，而且与轨道量子数$l$ (轨道形状)有关。

对于其中一个电子而言，其它$(N-1)$个电子的平均场对原子核有一定的屏蔽作用，该电子将感受到一个有效的核电荷数$Z^*$。不同的状态，径向概率分布不同，感受到的有效核电荷数也不同，即$Z^*$与该电子的状态量子数$n$, $l$有关。

$$E_{nl} = -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z_{nl}^{*2}}{n^2}$$
$$1 < Z_{nl}^* < Z$$

**体系总能量** $$E^{(0)} = \sum_{i=1}^{N} E_{n_i l_i}$$

## Figure & Layout Description

页面右侧包含一个径向概率分布图，整体布局为左文右图结构。图表采用直角坐标系，横轴标注为$r/a_0$（范围0-16），纵轴标注为$r^2 R_{nl}^2$（范围0.00-0.20）。图中包含三条曲线：
- 蓝色阴影区域表示1s轨道（标注"1s"），其概率密度峰值位于$r \approx 0$处，呈现尖锐单峰形态
- 红色实线表示2s轨道（标注"2s"），具有两个峰值（主峰在$r \approx 8$，次峰在$r \approx 2$）和一个节点
- 黑色实线表示2p轨道（标注"2p"），呈现单峰形态，峰值位于$r \approx 4$处

图中用红色文字标注"轨道贯穿"，指向1s与2p曲线在$r < 2$区域的交叠部分。坐标原点处有原子核示意图（紫色小球），1s轨道区域用浅蓝色半透明圆圈突出显示。所有曲线均采用平滑连续线条绘制，2p曲线峰值最高（约0.19），2s曲线次之（约0.18），1s曲线在原点处概率密度趋近于0。
