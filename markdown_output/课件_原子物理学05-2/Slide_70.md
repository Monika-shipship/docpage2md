# 电子组态能级的修正-- jj耦合

自旋-轨道相互作用远大于剩余静电势的情形

$|\hat{H}_2| >> |\hat{H}_1|$

$\hat{H} = \hat{H}_0 + \hat{H}_2 + \hat{H}_1$（红色禁止符号划去$\hat{H}_1$）$\approx \hat{H}_0 + \hat{H}_2$

$$\hat{H}_2 = \sum_{i=1}^{\nu} \zeta(r_i) \boldsymbol{l}_i \cdot \boldsymbol{s}_i$$

(1) 自旋-轨道相互作用引起的电子组态能级分裂

电子可以看作各自独立地在中心场中运动

由于自旋-轨道相互作用，每个电子的 $\boldsymbol{l}$ 和 $\boldsymbol{s}$ 耦合成单个电子的总角动量 $\boldsymbol{j}$

$$\boldsymbol{j} = \boldsymbol{l} + \boldsymbol{s}$$

> **ν个价电子组态**  
> $n_1 l_1, n_2 l_2, \cdots n_\nu l_\nu$
