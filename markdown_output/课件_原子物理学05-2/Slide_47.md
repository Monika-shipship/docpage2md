# 电子组态能级的修正-LS耦合

总自旋角动量满足

$$S^2 = S(S+1)\hbar^2$$

$$S_z = M_S\hbar$$

$$M_S = S, S-1, ..., -S$$

$S$是总自旋角动量量子数，$M_S$是相应的磁量子数。

$S$和$M_S$是好量子数。

交换能使电子组态能级按的$S$不同产生分裂。

$S$最大的能量最低
