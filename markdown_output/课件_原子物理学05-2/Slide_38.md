# 电子组态能级

## 电子组态能级的简并度

### 等效电子组态

设 $v$ 个电子是等效电子，电子组态是 $(nl)^v$，则该电子组态可能的状态数，即简并度为

$$ G = \frac{[2(2l + 1)]!}{v! [2(2l + 1) - v]!} $$

例如，氦原子激发态电子组态 $1s2s$，$G = 2 \times (2 \times 0 + 1) \times 2 \times (2 \times 0 + 1) = 4$  
(非等效电子组态)

基态电子组态 $1s^2$

$$ G = \frac{[2 \times (2 \times 0 + 1)]!}{2! \times [2 \times (2 \times 0 + 1) - 2]!} = 1 $$  
(等效电子组态)

例如：两个电子的 $n_1 = n_2 = 1$，$l_1 = l_2 = 0$，$m_{l1} = m_{l2} = 0$，  
则第四个量子数 $m_s$ 必须分别取 $+1/2$ 和 $-1/2$
