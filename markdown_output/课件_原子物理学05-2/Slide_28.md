## Figure & Layout Description

页面顶部是黑色粗体标题"多电子原子-元素周期律"，下方有灰色细横线分隔。标题下方左侧用红色字体标注"第五周期"。主体内容分两部分：  
1. **上半部分**：垂直排列4组元素信息（Rb/Sr/Y/Cd），每组包含元素符号、英文名、中文名、原子序数及基态电子组态。其中[Kr]用蓝色显示，电子层部分（如5s、4d等）用红色显示，Y和Cd条目之间有一个蓝色向下箭头连接，右侧用蓝色细线框标注"第二组过渡元素"。  
2. **下半部分**：横向排列的周期表片段表格，包含原子序数37-49的元素。表格分为两行：  
   - 第一行：元素符号（加粗）、英文名、原子量（小数点后保留2-3位）  
   - 第二行：对应电子组态（如4d¹5s²），其中Y、Nb、Mo等元素的电子组态用红色字体突出显示  
整体布局层次分明，颜色编码系统化：蓝色用于稀有气体核心电子层，红色用于价电子层，关键过渡元素用箭头和标注强调。表格边框为细实线，原子量数据精确到小数点后两位或三位。
