# 电子组态能级的修正-LS耦合

$\xi(L,S) > 0$

同一多重态的能级中，$J$越小，能量越低，这称为正常次序的多重态。

$\xi(L,S) < 0$

$J$越大，能量越低，这称为倒转次序的多重态。

C原子第一激发态 2p3p

## Figure & Layout Description

该幻灯片采用白底黑字的标准学术PPT布局，整体分为左右两部分。顶部是黑色粗体标题"电子组态能级的修正-LS耦合"，标题下方有一条细长的深灰色水平分割线（约2px粗），横跨整个页面宽度。

左侧区域为文本内容区，包含四段文字：
- 第一段：行内公式 $\xi(L,S) > 0$，使用标准数学字体
- 第二段：中文说明"同一多重态的能级中，$J$越小，能量越低，这称为正常次序的多重态。"，其中$J$为斜体变量
- 第三段：行内公式 $\xi(L,S) < 0$
- 第四段：中文说明"$J$越大，能量越低，这称为倒转次序的多重态。"，$J$为斜体
- 最底部是加粗标题"C原子第一激发态 2p3p"，作为右侧图表的说明

右侧区域为能级分裂示意图，采用垂直分层结构：
- 最左侧是基态标签"2p3p"，通过虚线连接到中间分裂层
- 中间层分为上下两部分：
  * 上部：单重态组，包含三条水平实线，从上到下依次标注$^1S$、$^1P$、$^1D$，右侧通过虚线连接到对应能级$^1S_0$、$^1P_1$、$^1D_2$
  * 下部：三重态组，包含三条水平实线，从上到下依次标注$^3S$、$^3P$、$^3D$，右侧通过虚线连接到对应能级
- 最右侧是$^3P$子项的详细分裂：
  * 用大括号标注"$^3P$"，右侧列出$J$值（2、1、0）
  * $J=2$对应能级$^3P_2$，标注能量值"0.005"
  * $J=1$对应能级$^3P_1$和$^3P_0$，标注能量值"0.0024"
  * $J=0$对应能级$^3P_0$
- 底部有三个水平标注：从左到右依次为$H_0$、$H_0+H_1$、$H_0+H_1+H_2$，对应能级分裂的不同阶段
- 所有能级线均为黑色实线，分裂过程用点状虚线表示，能量差值用水平短线标注

整个图表使用标准学术绘图规范，无彩色元素，所有文字均为黑色，能级线间距体现能量差比例（$^3P$子项分裂间距明显小于单重态与三重态之间的间距）。
