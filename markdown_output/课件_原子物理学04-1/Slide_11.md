# 原子的精细结构：电子的自旋

## 角动量空间取向量子化

$$L = \sqrt{l(l+1)}\hbar \quad L_z = m_l \hbar$$

对于给定量子数 $l$， $m_l = 0, \pm 1, \pm 2, \cdots \pm l$

$$\cos \theta = \frac{L_z}{|L|}$$

$l = 2$  
$|\mathbf{L}| = \sqrt{2(2+1)} \hbar$  
$m_l = 2\hbar, 1\hbar, 0, -1\hbar, -2\hbar$  
$$\cos \theta = \frac{0, \pm \hbar, \pm 2\hbar}{|L|}$$

Vector Model for Orbital Angular Momentum
