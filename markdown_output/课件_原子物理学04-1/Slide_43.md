## 总角动量

### 角动量相加的一般法则

设$J_1$和$J_2$是体系的两个角动量

| $J_1^2 = j_1(j_1 + 1)\hbar^2$ | $J_2^2 = j_2(j_2 + 1)\hbar^2$ |
| --- | --- |
| $J_{1z} = m_1\hbar$ | $J_{2z} = m_2\hbar$ |
| $m_1 = -j_1, -j_1+1, \cdots, j_1-1, j_1$ | $m_2 = -j_2, -j_2+1, \cdots, j_2-1, j_2$ |

两个角动量相加，得到总角动量

$$J = J_1 + J_2$$

大小  
$$J^2 = j(j+1)\hbar^2$$

z分量  
$$J_z = m\hbar$$

$j$、$m$是描述总角动量大小和z分量的量子数。
