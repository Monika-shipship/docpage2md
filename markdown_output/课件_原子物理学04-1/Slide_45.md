# 角动量相加

**问题**：已知体系角动量 $\mathbf{J}_1$ 和 $\mathbf{J}_2$ 的量子数 $j_1$ 和 $j_2$，总角动量量子数 $j$ 的可能取值是多少？

对于 $z$ 分量，有：
$$J_z = J_{1z} + J_{2z}$$
$$m = m_1 + m_2$$
$$|m|_{\text{max}} = j_1 + j_2$$
$$j_{\text{max}} = j_1 + j_2$$

右侧补充说明：
$$m_1 = -j_1, -j_1 + 1, \cdots, j_1 - 1, j_1$$
$$m_2 = -j_2, -j_2 + 1, \cdots, j_2 - 1, j_2$$

设 $j$ 的最小可能取值为 $j_{\text{min}}$，则
$$j = j_1 + j_2, \ j_1 + j_2 - 1, \ \ldots, \ j_1 - j_2 + 1, \ j_{\text{min}}$$
