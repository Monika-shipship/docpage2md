# 角动量相加

如果$J_1$和$J_2$彼此独立，没有耦合  
　　用量子数$(j_1, m_1, j_2, m_2)$来描述体系的状态。  

当$j_1$和$j_2$给定后，$m_1$和$m_2$分别有$(2j_1 + 1)$和$(2j_2 + 1)$个不同的取值，因此共有$(2j_1 + 1)(2j_2 + 1)$个不同的状态。  

如果$J_1$和$J_2$之间有耦合  
　　用量子数$(j_1, j_2, j, m)$来描述体系的状态  

当$j_1$和$j_2$给定后，有$(2j_1 + 1)(2j_2 + 1)$个不同的状态  
　　有$(2j_1 + 1)(2j_2 + 1)$组不同的$(j, m)$
