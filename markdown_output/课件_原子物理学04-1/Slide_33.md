# 自旋-轨道相互作用

## 原子光谱存在精细结构

- 左侧实验装置图：
  - Hydrogen discharge（氢放电管）
  - Vacuum chamber（真空室）
  - Rowland-grating（罗兰光栅）
  - Curved Photoplate（弯曲底片）
  - Pump（抽气泵）
  - 光路用红色线条表示，显示光线从放电管经光栅到弯曲底片

- 右侧能级图（$n$与$l$对应关系）：

| $n$ | $l=0$ | $l=1$ | $l=2$ | $l=3$ |
|-----|-------|-------|-------|-------|
| $\infty$ | | | | 电离极限 |
| 4 | 4s | 4p | 4d | 4f |
| 3 | 3s | 3p | 3d | |
| 2 | 2s | 2p | | |
| 1 | 1s | | | |

- H原子巴尔末线系：
  - $H_\alpha$线双重结构示意图（放大40,000倍）
  - 双重峰结构，标注"$H_\alpha$线双重结构"

1887年，迈克耳逊(A. Michelson)和莫雷(E. Morley)就利用高分辨的干涉光谱仪观测到了$H_\alpha$线的双重结构并测量到其间隔约为$0.253 \text{ cm}^{-1}$，双重结构的重心也偏离理论计算$0.2 \text{ cm}^{-1}$左右。

## Figure & Layout Description
页面顶部为黑色粗体标题"自旋-轨道相互作用"，下方有一条深灰色分隔线。左侧区域包含一个实验装置示意图：左上角标注"Hydrogen discharge"的放电管（内部红色点状表示放电），连接"Pump"抽气装置；右侧为"Vacuum chamber"真空室，内部有红色光路线条从放电管射向右侧弧形的"Rowland-grating"光栅，再反射至下方弧形的"Curved Photoplate"底片。装置图下方有一个水平波长轴（$\lambda$）的光谱示意图，显示多条红色竖线代表谱线。右侧区域分为上下两部分：上部为能级图，纵向排列主量子数$n$（1至$\infty$），横向按角量子数$l=0$~$3$分栏，标注$s$/$p$/$d$/$f$轨道能级，最上层虚线标注"电离极限"；下部为"H原子巴尔末线系"示意图，显示$H_\alpha$线的双重结构，包含一个放大40,000倍的双峰曲线图，横轴为频率$\nu$，标注"$H_\alpha$线双重结构"。页面底部为黑色正文，描述1887年实验细节。标题下有蓝色箭头符号"➢"引导子标题"原子光谱存在精细结构"。
