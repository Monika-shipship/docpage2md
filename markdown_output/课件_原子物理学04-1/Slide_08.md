# 量子数为什么是 $l(l+1)$？

## 角动量的本征方程：
$$\hat{L}^2 Y(\theta, \varphi) = \lambda \hbar^2 Y(\theta, \varphi)$$

## 波函数的标准条件：
$Y(\theta, \varphi)$ 在 $\theta \in [0, \pi]$ 有限；  
在 $\varphi \in [0, 2\pi]$ 单值。

## 则要求方程中的参数
$$\lambda = l(l+1), \quad l = 0,1,2,3,\ldots$$

## 定义：
$$L_+ = L_x + i L_y$$
$$L_- = L_x - i L_y$$
$$L^2 = \mathbf{L} \cdot \mathbf{L} = L_x^2 + L_y^2 + L_z^2$$

$$[L_z, L_+] = \hbar L_+$$
$$[L_z, L_-] = -\hbar L_-$$
$$[L_+, L_-] = 2\hbar L_z$$

$$[L^2, L_x] = [L^2, L_y] = [L^2, L_z] = [L^2, L_+] = [L^2, L_-] = 0$$
