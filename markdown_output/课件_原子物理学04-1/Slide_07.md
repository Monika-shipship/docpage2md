# 单电子(H)原子-角向方程

波函数的标准条件：  
$Y(\theta, \varphi)$ 在 $\theta \in [0, \pi]$ 有限；  
在 $\varphi \in [0, 2\pi]$ 单值。

则要求方程中的参数  
$$\lambda = l(l+1), \quad l = 0,1,2,3,\dots$$

方程的解是球谐函数  
$$Y_{lm}(\theta, \varphi) = N_{lm} P_l^m (\cos \theta) e^{im\varphi}$$  
$$l = 0,1,2,3,\dots \quad m = 0,\pm1,\pm2,\dots$$  
$$P_l^m(x) = \frac{1}{2^l l!} (1 - x^2)^{\frac{m}{2}} \frac{d^{l+m}}{dx^{l+m}} (x^2 - 1)^l \quad 0 < x < 1$$  
是特殊函数，称为连带勒让德多项式。
