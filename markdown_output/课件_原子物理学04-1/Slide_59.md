# 氢原子精细结构能级

氢原子或类氢离子的狄拉克方程在非相对论近似下：

$$\left[ \hat{H}_0 + \hat{H}_T + \hat{H}_{ls} + \hat{H}_V \right] u = Eu$$

$$\hat{H}_0 = \frac{\hat{p}^2}{2m_e} + V(r) = \frac{\hat{p}^2}{2m_e} - \frac{Ze^2}{4\pi\varepsilon_0 r}$$

$$\hat{H}_T = -\frac{\hat{p}^4}{8m_e^3 c^2}$$

$$\hat{H}_{ls} = \frac{1}{2m_e^2 c^2} \frac{1}{r} \frac{dV(r)}{dr} \boldsymbol{L} \cdot \boldsymbol{S}$$

$$\hat{H}_V = \frac{\hbar^2}{8m_e^2 c^2} \nabla^2 V(r) \quad \text{又称达尔文项}$$

$$E_{n,j} = E_n + \Delta E_{ls} + \Delta E_T + \Delta E_V = E_n - E_n \frac{\alpha^2 Z^2}{n^2} \left( \frac{3}{4} - \frac{n}{j + \frac{1}{2}} \right)$$
