# 跃迁和选择定则-电偶极跃迁

## 氢原子能级结构

- **左侧势能图**  
  - 纵轴：$E/\text{eV}$（能量单位：电子伏特）  
  - 能级标注：  
    - $n=1$：$-13.5$ eV  
    - $n=2$：$-3.37$ eV  
    - $n=3$：$-1.5$ eV  
    - $n=4$：$-0.84$ eV  
    - $n=\infty$：$0$ eV（电离极限）  
  - 势能公式：$E_p = -\frac{e^2}{4\pi\epsilon_0 r}$

- **右侧能级跃迁图**  
  - 主量子数 $n$ 分层：$n=1, 2, 3, 4, 5$  
  - 角量子数 $l$ 分组：  
    - $l=0$：$1s, 2s, 3s, 4s, 5s$  
    - $l=1$：$2p, 3p, 4p, 5p$  
    - $l=2$：$3d, 4d, 5d$  
    - $l=3$：$4f, 5f$  
    - $l=4$：$5g$  
  - 电离极限（ionization limit）：红色虚线标注  
  - 跃迁路径：  
    - 允许跃迁：绿色箭头（如 $2p \to 1s$）  
    - 禁戒跃迁：红色 "×" 标记（如 $2s \to 1s$）  
  - 图表标题：氢原子能级图（蓝色文字）

## Figure & Layout Description

该PPT页面采用上下结构布局。顶部为黑色粗体标题“跃迁和选择定则-电偶极跃迁”，下方有一条深灰色水平分隔线。主体区域分为左右两部分：  
- **左侧区域**：包含一个坐标图。纵轴标注“E/eV”，刻度从-13.5 eV到0 eV，黑色实线绘制势能曲线。曲线上方标注能级量子数 $n=1, 2, 3, 4, \infty$ 及对应能量值（-13.5, -3.37, -1.5, -0.84, 0 eV）。曲线右下方有黑色公式 $E_p = -\frac{e^2}{4\pi\epsilon_0 r}$，其中变量 $r$ 以箭头指向势能曲线右侧。  
- **右侧区域**：展示氢原子能级图。水平方向按主量子数 $n$ 分层（从下到上 $n=1$ 至 $n=5$），垂直方向按角量子数 $l$ 分组（从左到右 $l=0$ 至 $l=4$）。每个能级由黑色水平线表示，并标注轨道符号（如 $1s, 2s, 2p$ 等）。跃迁路径用彩色箭头表示：  
  - 绿色箭头：表示允许跃迁（如 $2p \to 1s$、$3d \to 2p$）  
  - 橙色箭头：表示另一类允许跃迁（如 $3p \to 2s$）  
  - 红色 "×" 标记：表示禁戒跃迁（如 $2s \to 1s$、$3s \to 1s$）  
  顶部有红色虚线标注“ionization limit”，右侧以蓝色文字标注“氢原子能级图”。整体图表以白色为背景，文字和线条清晰，颜色编码严格区分跃迁类型，层级分明地展示了量子态与跃迁规则的对应关系。
