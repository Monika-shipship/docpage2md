## 单电子(H)原子-角向方程

波函数的标准条件：  
$Y(\theta, \varphi)$ 在 $\theta \in [0, \pi]$ 有限；  
在 $\varphi \in [0, 2\pi]$ 单值。  

则要求方程中的参数  
$$\lambda = l(l+1), \quad l = 0,1,2,3,\dots$$  

方程的解是球谐函数  
$$Y_{lm}(\theta, \varphi) = N_{lm} P_l^m(\cos\theta) e^{im\varphi}$$  
$$l = 0,1,2,3,\cdots \quad m = 0,\pm1,\pm2,\cdots$$  

$$P_l^m(x) = \frac{1}{2^l l!}(1-x^2)^{\frac{m}{2}} \frac{d^{l+m}}{dx^{l+m}}(x^2-1)^l \quad 0 < x < 1$$  

是特殊函数，称为连带勒让德多项式。

## Figure & Layout Description

页面右侧包含一个三维直角坐标系示意图：  
- **坐标轴**：黑色实线表示x、y、z轴，箭头方向符合右手定则，x轴指向左下方，y轴水平向右，z轴垂直向上。  
- **向量与标注**：  
  - 蓝色实线向量 $\mathbf{r}$ 从原点指向第一卦限的黄色小球，标注 $|\mathbf{r}|$。  
  - 红色虚线从黄色小球垂直投影到xy平面，形成 $\mathbf{r}_x$（x轴方向）、$\mathbf{r}_y$（y轴方向）和 $\mathbf{r}_z$（z轴方向）的分量标注。  
  - 红色双向箭头标注角度：$\theta$ 为 $\mathbf{r}$ 与z轴的夹角，$\varphi$ 为xy平面内投影与x轴的夹角。  
- **颜色与层次**：坐标轴为黑色，角度标注为红色，向量 $\mathbf{r}$ 为蓝色，黄色小球作为位置标记，整体布局清晰区分几何元素与物理量。  
- **位置关系**：示意图位于页面右侧，与左侧文字内容形成左右分栏结构，文字部分通过数学公式和中文说明逐层展开理论推导。
