# 力学量的平均值、算符表示—平均值

**其它力学量呢？**

比如：粒子的位置 $\mathbf{r}$、动量 $\mathbf{p}$、动能 $T$、角动量 $\mathbf{L}$, ……

## (1) 粒子的位置 $\mathbf{r}$

**例如：一维无限深方势阱**

粒子的位置是不确定的，取值在 $[0, a]$ 之间。  
但粒子的概率分布是确定的，是

$$
|u(x)|^2 = 
\begin{cases}
\frac{2}{a} \sin^2 \frac{n\pi}{a} x, & 0 \leq x \leq a \\
0, & x < 0, \text{or}, x > a
\end{cases}
\quad n = 1, 2, 3, \ldots
$$

所以，可以得到粒子位置的平均值 (假设粒子处在基态 $n=1$ 态):

$$
\langle x \rangle = \int_0^a x |u(x)|^2 dx = \int_0^a x \frac{2}{a} \sin^2 \frac{\pi x}{a} dx = \frac{a}{2}
$$

加权平均
