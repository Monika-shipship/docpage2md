# 单电子(H)原子-径向方程

> 对于任何 $E > 0$，方程都有有意义的解。  
> 因此，在 $E > 0$ 时，我们得到的是连续谱。

> 对于 $E < 0$，电子束缚在库仑势阱内。  

做变换  
$$\rho = \left(-\frac{8m_e E}{\hbar^2}\right)^{1/2} r$$  
并设  
$$n = \frac{Ze^2}{4\pi\varepsilon_0 \hbar} \left(-\frac{m_e}{2E}\right)^{1/2}$$  
方程化为  
$$\left[\frac{d^2}{d\rho^2} - \frac{l(l+1)}{\rho^2} + \frac{n}{\rho} - \frac{1}{4}\right]\chi(\rho) = 0$$  
根据波函数标准条件的要求  
$$n = 1, 2, 3, \cdots \quad \text{且} \quad l \leq n-1$$
