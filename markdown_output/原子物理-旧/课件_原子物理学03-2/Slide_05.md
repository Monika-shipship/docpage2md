# 平均值计算（概率论）

若 $Y = g(X)$, 且 $P\{X = x_k\} = p_k$, $(k = 1,2,\cdots)$

则有 $$E(g(X)) = \sum_{k=1}^{\infty} g(x_k) p_k.$$

若 $X$ 是连续型的, 它的分布密度为 $f(x)$,

则有 $$E(g(X)) = \int_{-\infty}^{+\infty} g(x) f(x) dx.$$

加权平均 分布律和概率密度函数为“权”
