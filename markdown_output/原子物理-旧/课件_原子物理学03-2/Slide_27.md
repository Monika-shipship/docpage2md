# 单电子(H)原子-径向方程

$$R_{10}(r) = 2(Z / a_0)^{3/2} \exp(-Zr / a_0)$$
$$n = 1,2,3,\cdots$$
$$l = 0,1,2,\cdots,n-1$$
$$R_{20}(r) = 2(Z / 2a_0)^{3/2} (1 - Zr / 2a_0) \exp(-Zr / 2a_0)$$
$$R_{21}(r) = \frac{1}{\sqrt{3}} (Z / 2a_0)^{3/2} (Zr / a_0) \exp(-Zr / 2a_0)$$
$$R_{30}(r) = 2(Z / 3a_0)^{3/2} (1 - 2Zr / 3a_0 + 2Z^2 r^2 / 27a_0^2) \exp(-Zr / 3a_0)$$
$$R_{31}(r) = \frac{4\sqrt{2}}{9} (Z / 3a_0)^{3/2} (1 - Zr / 6a_0) (Zr / a_0) \exp(-Zr / 3a_0)$$
$$R_{32}(r) = \frac{4}{27\sqrt{10}} (Z / 3a_0)^{3/2} (Zr / a_0)^2 \exp(-Zr / 3a_0)$$
