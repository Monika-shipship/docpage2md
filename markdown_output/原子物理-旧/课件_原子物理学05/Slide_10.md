# 泡利不相容原理和交换效应—波函数的交换对称性

将两粒子再交换一次，有

$$P_{12}^2 \psi(q_1, q_2) = \lambda P_{12} \psi(q_1, q_2) = \lambda^2 \psi(q_1, q_2)$$

体系应回到原来的状态

$$\lambda^2 \psi(q_1, q_2) = \psi(q_1, q_2)$$

$$\lambda = \pm 1$$

λ = +1时  
$$P_{12} \psi(q_1, q_2) = \psi(q_1, q_2)$$  
体系的波函数对于交换两个粒子是对称的

λ = -1时  
$$P_{12} \psi(q_1, q_2) = -\psi(q_1, q_2)$$  
体系的波函数对于交换两个粒子是反对称的
