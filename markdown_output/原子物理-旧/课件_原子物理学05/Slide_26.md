# 泡利不相容原理和交换效应—交换效应

$$\hat{H}_0 u^{(0)}(\mathbf{r}_1, \mathbf{r}_2) = E^{(0)} u^{(0)}(\mathbf{r}_1, \mathbf{r}_2)$$

分离变量  
$$u^{(0)}(\mathbf{r}_1, \mathbf{r}_2) = u(\mathbf{r}_1)u(\mathbf{r}_2)$$

得到两个类氢离子的方程

$$\hat{h}_i u(\mathbf{r}_i) = E_i u(\mathbf{r}_i) \quad i = 1,2$$

$$\hat{h}_i = -\frac{\hbar^2}{2m} \nabla_i^2 - \frac{Ze^2}{4\pi\varepsilon_0 r_i}$$

化为单电子问题

设相应的波函数 $u_{n_i l_i m_i}(\mathbf{r}_i)$ 本征能量为  
$$E_{n_i} = -\frac{1}{2}mc^2\alpha^2 \frac{Z^2}{n_i^2}$$
