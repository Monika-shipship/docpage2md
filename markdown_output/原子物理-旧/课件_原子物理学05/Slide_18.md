# 泡利不相容原理和交换效应—两个电子的自旋波函数

引入总自旋角动量 $S$

$$S^2 = S(S+1)\hbar^2$$
$$S_z = M_s \hbar$$

总自旋角动量 $S$ 是两个电子的自旋 $s_1$ 和 $s_2$ 的矢量和

$$S = s_1 + s_2$$

单电子的自旋量子数

$$s_1 = s_2 = \frac{1}{2}$$

总自旋量子数 $S$ 的取值为

$$S = 0, 1$$

$S = 0$ 时 $M_s = 0$

$S = 1$ 时 $M_s = -1, 0, +1$
