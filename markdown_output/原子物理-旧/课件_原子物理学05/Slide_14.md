# 泡利不相容原理和交换效应—波函数的交换对称性

## 重新构造新的波函数

$$\psi_S(q_1, q_2) = \frac{1}{\sqrt{2}} \left[ \psi_\alpha(q_1)\psi_\beta(q_2) + \psi_\beta(q_1)\psi_\alpha(q_2) \right]$$ 交换对称

$$\psi_A(q_1, q_2) = \frac{1}{\sqrt{2}} \left[ \psi_\alpha(q_1)\psi_\beta(q_2) - \psi_\beta(q_1)\psi_\alpha(q_2) \right]$$ 交换反对称

$$1/\sqrt{2}$$ 是归一化因子
