# 原子的内层能级和特征X射线-X射线能谱

> 各元素的特征线波数的平方根与原子序数(原子核电荷数)成线性关系

**莫塞莱经验公式**（莫塞莱定律）

$K_{\alpha}$线  
$$\tilde{\nu}_K = R(Z-1)^2 \left( \frac{1}{1^2} - \frac{1}{2^2} \right)$$

$L_{\alpha}$线  
$$\tilde{\nu}_L = R(Z-7.4)^2 \left( \frac{1}{2^2} - \frac{1}{3^2} \right)$$

K线和L线的莫塞莱图

根据实验测量的特征线波数，从莫塞莱图上就可以标识元素的种类，所以特征谱又称为标识谱。
