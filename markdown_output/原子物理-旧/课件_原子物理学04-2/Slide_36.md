# 碱金属原子能级的超精细结构

## 以钠原子为例：

### 左侧能级图：
- **H (氢原子)**：显示 $n=2,3,4,5,\infty$ 能级，包含 $s,p,d,f$ 轨道
- **Na (钠原子)**：显示 $n=2,3,4,5,\infty$ 能级，包含 $s,p,d,f$ 轨道
- 能级图垂直方向为能量轴 $E(eV)$，范围从 $0$ 到 $-6\ eV$
- 图中显示电子从 $3s$ 基态向高能级的跃迁路径

### 右侧文字说明：

- ➤ 基态时价电子在 $3s$ 态，
  $l=0$，$s=1/2$，则 $j=1/2$
  $3^2S_{1/2}$
  没有精细结构分裂

- ➤ 价电子激发到 $ns$ ($n \ge 4$) 态，
  $n^2S_{1/2}$
  没有精细结构分裂

- ➤ 价电子激发到 $np$ ($n \ge 3$) 态，
  $l=1$，$s=1/2$，则 $j=3/2, 1/2$
  $n^2P_{3/2}$ 和 $n^2P_{1/2}$
  分裂为两个精细结构能级
  - $D_1(589.5930\ \text{nm})$
  - $D_2(588.9963\ \text{nm})$

- ➤ 价电子激发到 $nd$ ($n \ge 3$) 态，
  $l=2$，$s=1/2$，则 $j=5/2, 3/2$
  $n^2D_{5/2}$ 和 $n^2D_{3/2}$
  分裂为两个精细结构能级

### 底部标注：
严格单电子原子 ➔ 电子轨道贯穿 ➔ 精细结构

## Figure & Layout Description

图片整体布局分为三个主要区域：顶部标题区、中间内容区和底部标注区。

标题区位于图片最上方，以黑色粗体字显示"碱金属原子能级的超精细结构"，下方有一条深灰色横线作为分隔。

中间内容区分为左右两部分：
- 左侧是能级对比图，左侧标注"H"（氢原子），右侧标注"Na"（钠原子）。能级图垂直方向表示能量（E(eV)），从上到下能量从0 eV降至-6 eV。水平方向标有"S"、"P"、"D"、"F"表示不同轨道类型。主量子数n从2到∞标在右侧。图中显示了氢原子和钠原子的能级结构对比：氢原子能级为等间距排列，而钠原子能级在低n值时明显偏移（如3s能级低于氢原子对应能级），高n值时趋近于氢原子能级。钠原子能级图中用箭头表示从3s基态向高能级的跃迁路径。
- 右侧是文字说明区域，使用黑色和蓝色文字。每段说明前有蓝色箭头符号"➤"。文字内容详细解释了不同激发态下的能级结构情况，其中关键的能级符号如$3^2S_{1/2}$用红色突出显示，$n^2P_{3/2}$和$n^2P_{1/2}$用蓝色突出显示。右侧还包含一个能级分裂示意图，显示了不同n值下$^2S_{1/2}$、$^2P_{1/2,3/2}$、$^2D_{3/2,5/2}$、$^2F_{5/2,7/2}$等能级的分裂情况，其中np态分裂的两个能级分别标注为$D_1(589.5930\ \text{nm})$和$D_2(588.9963\ \text{nm})$。

底部标注区位于图片最下方，用蓝色文字标注"严格单电子原子 ➔ 电子轨道贯穿 ➔ 精细结构"，中间有两个蓝色箭头连接三个概念。

颜色使用方面：
- 标题为黑色粗体
- "以钠原子为例："为红色
- 能级图中的轨道名称"S"、"P"、"D"、"F"为黑色
- 文字说明中的箭头为蓝色
- 关键能级符号$3^2S_{1/2}$为红色，$n^2P_{3/2}$和$n^2P_{1/2}$为蓝色
- 底部标注为蓝色
