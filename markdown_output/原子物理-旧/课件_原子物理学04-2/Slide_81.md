## Figure & Layout Description
图像顶部有黑色粗体标题"原子磁力仪装置"，下方有一条深灰色水平分隔线。主体为从左至右的系统框图：  
1. **左侧区域**：黑色边框矩形框标注"抽运激光器及其稳频系统"，右侧通过红色粗箭头连接至"声光调制器 AOM"模块。  
2. **中间光路**：  
   - 红色实线表示光路，依次穿过AOM、黑色椭圆形"λ/4玻片"、双凸透镜组成的"扩束透镜组"。  
   - 光路进入中心灰色矩形"铷泡"区域，该区域被多层同心圆环包围，外层标注"亥姆霍兹线圈"，内层标注"铷泡加热模块"。  
   - 铷泡上方有黑色椭圆"偏振片"，顶部垂直红色光路连接"探测激光器"。  
   - 磁场方向用黑色箭头标注$B$，水平向右穿过铷泡。  
3. **右侧探测部分**：  
   - 光路经PBS分束后，红色箭头指向"光电探测器"，探测器输出连接倒三角形符号的"差分放大电路"。  
4. **底部控制回路**：  
   - 黑色细线构成闭环控制：DSP模块连接"信号源"和"PCI-5922"，PCI-5922连接"PC"，PC反馈至DSP。  
   - 所有文字标注均为黑色，关键光路用红色突出，模块框为黑色细边框矩形，连接线分红色（光路）和黑色（信号线）两类。
