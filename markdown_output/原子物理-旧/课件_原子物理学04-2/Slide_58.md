# 外磁场中的原子

## $np$能级的塞曼分裂随磁场的关系

### 能级分裂数据表

| $m_l$ | $m_s$ | $\Delta E$ |
|-------|-------|------------|
| +1    | +1/2  | $2\mu_B B + \xi_{np}/2$ |
| 0     | +1/2  | $\mu_B B$ |
| +1    | +1/2 或 -1/2 | $-\xi_{np}/2$ |
| 0     | -1/2  | $-\mu_B B$ |
| -1    | -1/2  | $-2\mu_B B + \xi_{np}/2$ |

### 磁场区域特性

- **零场**  
  自旋-轨道作用  
  精细结构能级

- **弱场**  
  自旋-轨道作用主导  
  精细结构能级磁分裂

- **强场**  
  自旋-轨道耦合被破坏  
  能级按$m_l$、$m_s$磁分裂

## Figure & Layout Description

图片整体采用学术PPT标准布局，白色背景。顶部有黑色粗体主标题"外磁场中的原子"，下方有灰色细横线分隔。横线下方是红色副标题"$np$能级的塞曼分裂随磁场的关系"。

左侧为能量-磁场关系图：垂直Y轴标记为"$E$"（能量轴），原点处标注两个初始能级"$^2P_{3/2}$"（上方）和"$^2P_{1/2}$"（下方）。从这两个能级向右延伸出多条黑色实线，表示不同磁场强度下的能级分裂轨迹：
- 上半部分：从$^2P_{3/2}$延伸出三条向上倾斜的直线
- 下半部分：从$^2P_{1/2}$延伸出三条向下倾斜的直线
- 中间区域：有两条水平线表示特定分裂状态

右侧为数据表格区域，包含三列：
- 第一列"$m_l$"：磁量子数，数值为+1、0、+1（重复）、0、-1
- 第二列"$m_s$"：自旋磁量子数，数值为+1/2、+1/2、{+1/2或-1/2}、-1/2、-1/2
- 第三列"$\Delta E$"：能量分裂表达式，含$\mu_B$（玻尔磁子）和$\xi_{np}$（自旋-轨道耦合系数）

底部有三个水平分区：
- 左侧"零场"：蓝色文字，下方两行黑色小字说明
- 中间"弱场"：蓝色文字，下方两行黑色小字说明，配有橙色向上箭头指向图表中间区域
- 右侧"强场"：蓝色文字，下方两行黑色小字说明，配有橙色向上箭头指向图表右侧区域

图表中关键特征：
- 橙色箭头从底部"弱场"和"强场"区域指向图表中的特定分裂线
- "强场"区域旁有特殊标注"$B \to 1$"表示磁场增强方向
- 所有数学符号均使用标准物理符号格式，下标清晰（如$\xi_{np}$）
