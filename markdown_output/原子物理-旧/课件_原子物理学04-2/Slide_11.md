# 氢原子能级的超精细结构

对于氢原子(或类氢离子)，量子力学计算得到

$$a = g_I \left( \frac{m_e}{M_p} \right) m_e c^2 \alpha^4 \frac{1}{j(j+1)(2l+1)} \frac{Z^3}{n^3}$$

$$= -2g_I \left( \frac{m_e}{M_p} \right) \frac{\alpha^2 Z}{n} E_n \frac{1}{j(j+1)(2l+1)}$$

精细结构相互作用能  
$$\propto \alpha^2 E_n$$

超精细结构相互作用能  
$$\propto \left( \frac{m_e}{M_p} \right) \alpha^2 E_n$$
