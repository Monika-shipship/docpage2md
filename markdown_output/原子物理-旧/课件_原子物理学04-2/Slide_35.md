# 碱金属原子能级的超精细结构

对于 $l=0$ 的 $S$ 态能级没有精细结构分裂；

对于 $l \neq 0$ 的态，价电子的自旋量子数 $s = \frac{1}{2}$，  
总角动量量子数 $j = l \pm \frac{1}{2}$，

相应的能级将按照 $j$ 量子数的不同分裂为两个精细结构能级，  
能量的修正值分别为

$$
\Delta E_{ls} = 
\begin{cases} 
\dfrac{\mu c^2 \alpha^4 Z_{fs}^{*4}}{2 n^{*3}} \dfrac{1}{(l+1)(2l+1)} > 0, & j = l + \dfrac{1}{2} \\[2em]
-\dfrac{\mu c^2 \alpha^4 Z_{fs}^{*4}}{2 n^{*3}} \dfrac{1}{l(2l+1)} < 0, & j = l - \dfrac{1}{2}
\end{cases}
$$

能级间的裂距为

$$
\delta E_{ls} = \Delta E_{ls}\left(j = l + \frac{1}{2}\right) - \Delta E_{ls}\left(j = l - \frac{1}{2}\right) = \dfrac{\mu c^2 \alpha^4 Z_{fs}^{*4}}{2 n^{*3}} \dfrac{1}{l(l+1)}
$$
