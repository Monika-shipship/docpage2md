# 碱金属原子的能级与光谱—碱金属原子光谱

## 能级结构

- **坐标轴**：左侧纵轴表示能量 $E(\text{eV})$，范围从 $-6$ eV 到 $0$ eV
- **原子类型**：从左到右分为三个区域，分别表示 Li、Na 和 H 原子
- **量子数标注**：
  - 每个区域顶部标注 $S$、$P$、$D$、$F$（表示角量子数 $l = 0, 1, 2, 3$）
  - 右侧纵轴标注主量子数 $n$，从 $2$ 到 $\infty$
- **能级线**：
  - Li 区域：显示 $2s$、$2p$、$3s$、$3p$、$3d$、$4s$、$4p$、$4d$、$4f$、$5s$、$5p$、$5d$、$5f$ 等能级
  - Na 区域：显示 $3s$、$3p$、$3d$、$4s$、$4p$、$4d$、$4f$、$5s$、$5p$、$5d$、$5f$ 等能级
  - H 区域：显示 $2s$、$2p$、$3s$、$3p$、$3d$、$4s$、$4p$、$4d$、$4f$、$5s$、$5p$、$5d$、$5f$ 等能级
- **特殊标记**：
  - $n = \infty$ 处有斜线阴影区域，表示电离极限
  - 图下方标注"钠原子能级图"

## Figure & Layout Description

该图是一个碱金属原子能级对比示意图，整体布局为横向排列的三个能级结构图，从左到右依次为锂(Li)、钠(Na)和氢(H)原子的能级分布。

**整体结构**：
- 顶部有大标题"碱金属原子的能级与光谱—碱金属原子光谱"，使用粗体黑体字，下方有一条水平分隔线
- 图表主体占据页面中央大部分区域，呈矩形布局
- 底部居中位置有小字标注"钠原子能级图"

**坐标系统与标注**：
- 左侧垂直坐标轴明确标注"$E(\text{eV})$"，刻度从$-6$ eV到$0$ eV，每1 eV一个主刻度
- 右侧垂直坐标轴标注主量子数"$n$"，标记有$2$、$3$、$4$、$5$和$\infty$
- 三个原子区域顶部均标注"$S$ $P$ $D$ $F$"，表示不同角量子数的轨道类型

**能级图分区细节**：
1. **Li区域（左侧）**：
   - 顶部标注"Li"
   - 能级从上到下依次为：$5s$、$5p$、$5d$、$5f$；$4s$、$4p$、$4d$、$4f$；$3s$、$3p$、$3d$；$2s$、$2p$
   - $2s$能级位置最低，约在$-5.4$ eV处
   - 能级线向右上方倾斜，表示随主量子数增加能量升高

2. **Na区域（中间）**：
   - 顶部标注"Na"
   - 能级结构与Li类似但位置不同
   - $3s$能级位置最低，约在$-5.1$ eV处
   - 能级线同样向右上方倾斜，但倾斜角度与Li区域不同

3. **H区域（右侧）**：
   - 顶部标注"H"
   - 能级呈现规则排列，相同$n$值的不同$l$能级（如$2s$和$2p$）处于同一水平线
   - 能级间距随$n$增大而减小，符合氢原子能级公式特征

**共同特征与视觉元素**：
- $n = \infty$处有一条水平虚线，上方有斜线阴影区域，表示电离极限（能量为$0$ eV）
- 有细实线连接不同能级，表示可能的电子跃迁路径
- 所有能级线均为黑色实线，背景为纯白色
- 三个区域的能级线在垂直方向上大致对齐，便于比较不同原子的能级结构差异
- 氢原子(H)区域显示$2s$与$2p$能级简并，而Li和Na区域中相同$n$值但不同$l$的能级能量不同（如$3s$、$3p$、$3d$能量依次升高）

**特殊细节**：
- 钠原子的$3s$能级明显低于其他同$n$值能级，是钠原子基态
- 氢原子能级间距随$n$增大而减小，符合$E_n \propto -\frac{1}{n^2}$规律
- Li和Na区域中，相同$n$值的$s$轨道能级最低，$f$轨道能级最高
