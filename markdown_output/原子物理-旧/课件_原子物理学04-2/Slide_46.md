# 外磁场中的原子—弱磁场

由此得到

$$\mu_j = -\frac{\mu_B}{\hbar} \left(1 + \frac{J^2 + S^2 - L^2}{2J^2}\right) J$$

代入 $J^2 = j(j+1)\hbar^2, L^2 = l(l+1)\hbar^2, S^2 = s(s+1)\hbar^2$

得

$$\mu_j = -\frac{\mu_B}{\hbar} \left[1 + \frac{j(j+1) + s(s+1) - l(l+1)}{2j(j+1)}\right] J$$

$$\mu_j = -g_j \frac{\mu_B}{\hbar} J$$

其中

$$g_j = 1 + \frac{j(j+1) + s(s+1) - l(l+1)}{2j(j+1)}$$

是朗德(Landé)g因子，它反映了原子总角动量和原子磁矩的回磁比。
