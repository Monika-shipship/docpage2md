# 碱金属原子的能级与光谱—碱金属原子光谱

## 第一辅线系(The first subordinate series)，又称漫线系(The diffuse series)

$nd \rightarrow 3p$

$$\tilde{\nu}_n = \frac{R}{(3-\Delta_p)^2} - \frac{R}{(n-\Delta_d)^2} \quad n = 3, 4, 5, \ldots$$

$\Delta_d$是$l=2$的$d$能级的量子数亏损。

## 第二辅线系(The second subordinate series)，又称锐线系(The sharp series)

$ns \rightarrow 3p$

$$\tilde{\nu}_n = \frac{R}{(3-\Delta_p)^2} - \frac{R}{(n-\Delta_s)^2} \quad n = 4, 5, 6, \ldots$$

## 伯格曼线系(The Bergmann series)，又称基线系(The fundamental series)

$nf \rightarrow 3d$

$$\tilde{\nu}_n = \frac{R}{(3-\Delta_d)^2} - \frac{R}{(n-\Delta_f)^2} \quad n = 4, 5, 6, \ldots$$

$\Delta_f$是$l=3$的$f$能级的量子数亏损。
