# 塞曼谱线的偏振特性

**塞曼分裂的各谱线均是偏振的**

$Δm_j = 0, ±1$

$Δm_l = 0, ±1$

页面顶部为黑色粗体标题"塞曼谱线的偏振特性"，下方有蓝色副标题"塞曼分裂的各谱线均是偏振的"。主体内容分为左右两个对称的能级分裂示意图：

左侧图示：
- 垂直方向排列三个能级组：顶部为$3^2P_{3/2}$（四重分裂），中间为$3^2P_{1/2}$（双重分裂），底部为$3^2S_{1/2}$（双重分裂）

- 底部标注$D_1$和$D_2$谱线区域，包含$\bar{σ}π|π\bar{σ}^+$和$σ\bar{σ}|ππ|σ^+\bar{σ}^+$等偏振标记

右侧图示：
- 垂直方向排列两个能级组：顶部为$3p$（五重分裂），底部为$3s$（双重分裂）
