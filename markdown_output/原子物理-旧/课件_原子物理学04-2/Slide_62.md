# 塞曼谱线的偏振特性

$ \Delta m = m_1 - m_2 = 0 $

原子发射一个光子后，在磁场方向的角动量不变。

发射的光子的角动量必定垂直于磁场。

相应电磁波的电矢量一定在与磁场平行的平面内，又由于x、y轴取向的任意性，电矢量x和y分量的平均值均为零，所以平均而言电矢量只有磁场方向的z分量。

对应于在z方向的电偶极振荡，称为π偏振态。

**纵向观测：** 无

**横向观测：** 观测到电矢量在磁场方向上的线偏振光。
