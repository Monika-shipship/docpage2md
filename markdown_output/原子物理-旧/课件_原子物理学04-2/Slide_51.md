# 外磁场中的原子—弱磁场

以钠原子为例： **D双线的塞曼分裂**

对于磁量子数，电偶极跃迁要满足**选择定则**

$$\Delta m = 0, \pm 1$$

$$\tilde{\nu} = \tilde{\nu}_0 + (m_2 g_2 - m_1 g_1) \mathcal{L}$$

**$D_1$** 偏离原谱线位置

$$-\frac{4}{3}\mathcal{L}, -\frac{2}{3}\mathcal{L}, +\frac{2}{3}\mathcal{L}, +\frac{4}{3}\mathcal{L}$$

**$D_2$** 偏离原谱线位置

$$-\frac{5}{3}\mathcal{L}, -\mathcal{L}, -\frac{1}{3}\mathcal{L}, +\frac{1}{3}\mathcal{L}, +\mathcal{L}, +\frac{5}{3}\mathcal{L}$$

## Figure & Layout Description

PPT页面采用左右分栏布局，左侧为文字说明区，右侧为能级分裂示意图。

标题"外磁场中的原子—弱磁场"位于页面顶部，黑色粗体字，下方有一条深灰色横线作为分隔。副标题"以钠原子为例：D双线的塞曼分裂"中，"以钠原子为例："为红色字体，"D双线的塞曼分裂"为蓝色字体，形成视觉重点。

左侧文字区域：
- "对于磁量子数，电偶极跃迁要满足选择定则"段落中，"选择定则"四个字以红色加粗显示
- 选择定则公式$\Delta m = 0, \pm 1$以黑色行间公式呈现
- 波数变化公式$\tilde{\nu} = \tilde{\nu}_0 + (m_2 g_2 - m_1 g_1) \mathcal{L}$为黑色行间公式
- "$D_1$ 偏离原谱线位置"中，"D₁"为红色加粗字体，下方列出四个偏离位置公式
- "$D_2$ 偏离原谱线位置"中，"D₂"为红色加粗字体，下方列出六个偏离位置公式

右侧能级图：
- 垂直排列三个能级：顶部为$3^2P_{3/2}$，中间为$3^2P_{1/2}$，底部为$3^2S_{1/2}$
- 每个能级向右延伸，显示分裂后的子能级，右侧标注m值和mg值两列
- $3^2P_{3/2}$能级分裂为4个子能级，m值分别为+3/2、+1/2、-1/2、-3/2，对应mg值分别为+2、+2/3、-2/3、-2
- $3^2P_{1/2}$能级分裂为2个子能级，m值分别为+1/2、-1/2，对应mg值分别为+1/3、-1/3
- $3^2S_{1/2}$能级分裂为2个子能级，m值分别为+1/2、-1/2，对应mg值分别为+1、-1
- 垂直箭头表示电偶极跃迁，左侧一组跃迁标注为$D_1$（红色），右侧一组标注为$D_2$（红色）
- 图表底部有光谱线分布示意图，标注了$\sigma$和$\pi$偏振态，$D_1$区域标注有"$\tilde{\sigma} \pi | \pi \tilde{\sigma}^+$"，$D_2$区域标注有"$\tilde{\sigma} \tilde{\sigma} | \pi \pi | \tilde{\sigma}^+ \tilde{\sigma}^+$"，并以箭头表示波数$\nu$增加方向

整个页面背景为白色，文字主要为黑色，关键术语用红色或蓝色突出显示，能级图使用黑色实线表示能级，黑色箭头表示跃迁，红色标注突出重点。
