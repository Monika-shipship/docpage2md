# H原子光谱和Bohr原子模型

## (2) 频率条件

如何产生光谱辐射？

从Ritz公式出发 $\tilde{\nu} = T(m) - T(n)$

$$T(n) = \frac{R_H}{n^2}$$

$$hc\tilde{\nu} = h\nu = hcT(m) - hcT(n)$$

$$E_n = -hcT(n)$$

Einstein的光量子

$$= E_n - E_m$$

定态m的能量

定态n的能量

定态跃迁：原子能量的改变是从一个定态跃迁(突然变到)到另一个定态。

跃迁过程放出(或吸收)一个光子，其能量：

$$h\nu = |E_m - E_n|$$

Bohr频率规则

$$\nu = |E_m - E_n| / h$$
