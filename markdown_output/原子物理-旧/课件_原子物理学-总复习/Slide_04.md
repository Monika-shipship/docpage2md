# 我个人的理解

- 原子物理叙事方式与其他课程不同
  - 更多的是历史认识过程的介绍，好处在于可以让学生了解整个近代物理的发展过程和量子理论的不断完善经历；
  - 缺点：1. 整个课程不像其他普通物理那样有严谨的公式推导，更多的是靠物理学家们的思想风暴利用经典图像和经验公式建立模型，并根据实验结果对模型不断完善的过程（在学完量子力学后会有不同认识）；2. 有些结论会根据模型不同不断改变和完善；知识点太多，而且课时有限。
- 我个人讲课的特点
  - 多引入自己了解的与原子物理相关的科研前沿，期望能启发你们的兴趣
  - 期望能尽量的把历史背景交代清楚，能让你们对当时的历史背景和重大科学发现的过程交代清楚，说不定这样的思维就能培养1-2名和他们一样伟大的物理学家
  - 尽可能地在自己的知识范围内讲清楚来龙去脉
  - 量子力学内容讲的稍微多了一些，量子力学的公式稍微多了一些，主要为了帮助理解，但不过多要求，不用掌握（这部分主要参考中科大《近代物理学》教材）
