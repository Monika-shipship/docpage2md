# He原子的光谱与能级

## 氢原子(类氢离子)
- 中心：$+Ze, M$
- 电子：$-e, m_e$

## 碱金属原子
- 中心：$+Ze$
- 有效屏蔽电荷：$-(Z-1)e$
- 脱离电子：$-e$

## 氦原子
- 中心：$+2e$
- 电子：$-e$（两个）

## Figure & Layout Description
页面顶部为黑色粗体标题"He原子的光谱与能级"，下方有深灰色水平分割线。主体内容由三个从左至右排列的原子结构示意图组成，各图之间用蓝色右向箭头连接。  
1. **左侧图（氢原子）**：蓝色细线圆环代表电子轨道，中心红色实心圆标注红色文字"$+Ze, M$"，轨道上绿色小点标注红色文字"$-e, m_e$"。底部蓝色文字标注"氢原子(类氢离子)"。  
2. **中间图（碱金属原子）**：浅蓝色填充的大圆代表电子云，中心红色实心圆标注红色文字"$+Ze$"，电子云内部标注红色文字"$-(Z-1)e$"。右上角有蓝色小球（脱离的电子）及红色文字"$-e$"。底部蓝色文字标注"碱金属原子"。  
3. **右侧图（氦原子）**：深蓝色细线圆环代表电子轨道，中心红色实心圆标注红色文字"$+2e$"，轨道上有两个蓝色小球（电子）及红色文字"$-e$"。底部蓝色文字标注"氦原子"。  
所有电荷标注均使用红色字体，原子核用红色实心圆表示，电子用蓝色/绿色小球表示，箭头为深蓝色立体样式。
