# 外磁场中的原子

*np能级的塞曼分裂随磁场的关系*

| $m_l$ | $m_s$ | $\Delta E$ |
|--------|--------|-------------|
| +1 | +1/2 | $2\mu_B B + \xi_{np}/2$ |
| 0 | +1/2 | $\mu_B B$ |
| -1, +1 | +1/2, -1/2 | $-\xi_{np}/2$ |
| 0 | -1/2 | $-\mu_B B$ |
| -1 | -1/2 | $-2\mu_B B + \xi_{np}/2$ |

零场  
自旋-轨道作用  
精细结构能级  

弱场  
自旋-轨道作用主导  
精细结构能级磁分裂  

强场  
自旋-轨道耦合被破坏  
能级按$m_l$、$m_s$磁分裂
