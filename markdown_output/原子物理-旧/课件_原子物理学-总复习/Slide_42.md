# 多重态的符号表示形式（P148）

## 四个量子数（$n, s, l, j$）

$$n^{2s+1}X_j$$

$l = 0, 1, 2, 3, \dots$

$X \rightarrow S、P、D、F、\dots$

$n=3, s=1/2, l=1, j=1/2、3/2$,相应的写为：

$$3^2P_{1/2}$$ $$3^2P_{3/2}$$

$3^2D_{3/2}、2^2P_{1/2}$分别代表什么呢？
