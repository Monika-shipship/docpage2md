# Slide 3

- 原子物理是探索物质在微观层面的规律的课。其实“微观”与“宏观”只是相对于人类的尺度而言，在某种更广泛的意义上，星系之大与原子之小的尺度差异，是否微乎其微呢？
- 对整个物理学来说，研究物理就是研究世界的运行规律，原子物理和其他专业课都是相应的更细划分，有的课可以作为理解原子物理的基础，如经典光学；有的课是在原子物理基础上的更深探究，如量子力学量子光学
- 最大收获是对思维的锻炼，量子层面的实验现象与结论是很反直觉的
- 困难:心态上有时会浮躁，不能潜心钻研
- 建议:可以适当讲一些前沿发现

1. 是一门充当着承接经典，开启量子角色的课程。这门课也是物理思想最为爆炸的一门课之一，能够了解到以前物理学家思考问题的方式以及从无到有的思维。
2. 与量子力学的关联是必然的，也与经典力学有这很大关系(毕竟很多结论推导离不开牛顿第二定律)，以及数理方法等等。相通之处就是物理模型的建立和物理问题的分析。
3. 了解了量子与经典的区别;了解了先辈思考的方式;明白了宏观与微观的差距。
4. 一些概念理解的困难;量子数的物理含义不明确;前后联系不紧密
5. 抓住主要历史事件(比如实验)进行详细讲述，一些不严谨的公式推导部分略去，重点讲物理分析与物理思维。
