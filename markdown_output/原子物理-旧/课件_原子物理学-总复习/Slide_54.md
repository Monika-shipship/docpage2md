# 外磁场中的原子—弱磁场

以钠原子为例： **D双线的塞曼分裂**

对于磁量子数，电偶极跃迁要满足**选择定则**

$$\Delta m = 0, \pm 1$$

$$\tilde{\nu} = \tilde{\nu}_0 + (m_2 g_2 - m_1 g_1) \mathcal{L}$$

**$D_1$** 偏离原谱线位置

$$-\frac{4}{3}\mathcal{L}, -\frac{2}{3}\mathcal{L}, +\frac{2}{3}\mathcal{L}, +\frac{4}{3}\mathcal{L}$$

**$D_2$** 偏离原谱线位置

$$-\frac{5}{3}\mathcal{L}, -\mathcal{L}, -\frac{1}{3}\mathcal{L}, +\frac{1}{3}\mathcal{L}, +\mathcal{L}, +\frac{5}{3}\mathcal{L}$$

## Figure & Layout Description

页面顶部为黑色粗体标题"外磁场中的原子—弱磁场"，下方有深蓝色横线分隔。正文区域分为左右两部分：左侧为文字说明区，右侧为能级分裂示意图。

文字区采用多级排版：
- 红色文字"以钠原子为例："后接蓝色加粗文字"D双线的塞曼分裂"
- 黑色常规文字说明"对于磁量子数，电偶极跃迁要满足"后接红色加粗"选择定则"
- 数学公式区域包含三个核心公式，其中$\Delta m$选择定则用行间公式展示，频率偏移公式和分裂位置公式均采用行间公式排版
- $D_1$和$D_2$标注为红色加粗，其后跟随黑色常规文字"偏离原谱线位置"

右侧示意图包含：
- 三层能级结构：顶部$3^2P_{3/2}$（4个子能级）、中部$3^2P_{1/2}$（2个子能级）、底部$3^2S_{1/2}$（2个子能级）
- 每个能级右侧标注$m$和$mg$值，其中$3^2P_{3/2}$对应$m=+3/2, +1/2, -1/2, -3/2$和$mg=+2, +2/3, -2/3, -2$
- 红色标注的$D_1$和$D_2$跃迁组，分别用6条和10条跃迁线表示（实线和虚线区分）
- 底部光谱示意图标注$\sigma^-, \pi, \sigma^+$偏振态，横轴标记为$\nu$
- 所有能级线采用黑色实线，分裂子能级用黑色虚线连接

整体布局采用左文右图结构，文字区使用黑/红/蓝三色强调关键概念，示意图通过垂直排列的能级线和水平跃迁线清晰展示塞曼分裂过程。
