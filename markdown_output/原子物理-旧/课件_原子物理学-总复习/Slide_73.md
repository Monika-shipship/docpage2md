# L-S耦合

由给定 $L$、$S$ 值决定的能级关于 $M_L$ 和 $M_S$ 仍是简并的，简并度为 $(2L+1)(2S+1)$。

在原子光谱学中将这简并的 $(2L+1)(2S+1)$ 个态的集合称为原子的多重态或多重项，简称项(term)或谱项，记作 $^{2S+1}L$，$2S+1$ 称为谱项的多重数。大写字母 $S$、$P$、$D$、$F$ 等表示 $L = 0, 1, 2, \dots$ 的状态。

页面整体采用白色背景，黑色文字的学术PPT风格。标题"L-S耦合"位于页面顶部左侧，使用36号加粗黑体字，下方有一条贯穿页面宽度的深灰色水平分隔线（约2px粗）。正文部分包含两段标准16号宋体文字，首行无缩进，行间距适中。

页面下半部分包含一个能级分裂示意图：
- 左侧起始点标注"2p3p"，位于一条水平实线的左端
- 从"2p3p"向右延伸出两条主要分支：
  * 上分支：用虚线连接至三个水平能级线，从上至下依次标注为$^1S$、$^1P$、$^1D$（上标数字为1，字母为大写），各能级线长度一致，间距相等
  * 下分支：用点线连接至三个水平能级线，从上至下依次标注为$^3S$、$^3P$、$^3D$（上标数字为3），能级线长度与上分支相同但垂直位置更低
- 能级线之间用细虚线连接表示简并关系：上分支的$^1S$-$^1P$-$^1D$用虚线连接，下分支的$^3S$-$^3P$-$^3D$用点线连接
- 页面底部有两组标签：
  * 左侧标注"$H_0$"，对应"2p3p"原始能级位置
  * 右侧标注"$H_0+H_1$"，对应分裂后的能级组
