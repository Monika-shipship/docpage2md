# 外磁场中的原子—强磁场

**以钠原子为例：能级的塞曼分裂**

考察 $3p$ 和 $3s$ 能级

**3s**  
$m_l = 0, m_s = \pm \frac{1}{2}$  

$\Delta E = (m_l + 2m_s)\mu_B B$  
$= (0 \pm 2 \times \frac{1}{2})\mu_B B = \pm \mu_B B$

**3p**  
$m_l = 0, \pm 1; m_s = \pm \frac{1}{2}$  

$\Delta E = (m_l + 2m_s)\mu_B B$  
$= 0, \pm \mu_B B, \pm 2\mu_B B$

### 3p 能级分裂对应量子数

| $m_l$   | $m_s$     |
|---------|-----------|
| +1      | +1/2      |
| 0       | +1/2      |
| (+/-)1  | (-/+)1/2  |
| 0       | -1/2      |
| -1      | -1/2      |

### 3s 能级分裂对应量子数

| $m_l$ | $m_s$ |
|-------|-------|
| 0     | +1/2  |
| 0     | -1/2  |
