# 外磁场中的原子—强磁场

以钠原子为例： *D线的塞曼分裂*

6个允许跃迁两两简并，得到3条谱线。

德国物理学家帕邢(F. Paschen)和巴克(E. Back)最早讨论了这种强磁场破坏自旋-轨道耦合的情形，所以又称为帕邢-巴克(Paschen-Back)效应。

$$\tilde{\nu} = \tilde{\nu}_0 + \frac{\mu_B B}{hc} \Delta m_l = \begin{cases} \tilde{\nu}_0 + \mathcal{L}, & \Delta m_l = +1 \\ \tilde{\nu}_0, & \Delta m_l = 0 \\ \tilde{\nu}_0 - \mathcal{L}, & \Delta m_l = -1 \end{cases}$$

洛伦兹用经典的“电子论”得到的就是上述结果。

$$\mathcal{L} = \frac{\mu_B B}{hc} = \frac{eB}{4\pi m_e c}$$

1897年塞曼推算出了电子的荷质比。
