# j-j耦合

相同原子的能级系列中，随激发电子 $n$ 增大，能级趋向于 $jj$ 耦合。这是因为当两个电子中的一个电子趋向高激发态时，两个电子之间的静电相互作用迅速减小，静电非中心力也迅速减小。

## Figure & Layout Description

该幻灯片采用白底黑字设计，整体布局分为三个垂直区域：标题区、文本区和图示区。标题区位于顶部，包含大号加粗黑体标题“j-j耦合”，下方有一条贯穿页面宽度的深灰色水平分隔线。文本区位于标题线下方，包含一段左对齐的中文说明文字，使用标准宋体字，行距适中。

图示区占据页面下半部分，是一个能级演化示意图，整体呈水平排列结构，从左到右分为三个子区域：
1. **左侧区域**：顶部标注“$LS$ 耦合”（中文）和“$L$-$S$ coupling”（英文），垂直排列四个水平能级线（实线），从上至下依次标注为 $^1P_1$、$^3P_2$、$^3P_1$、$^3P_0$，底部对应标签“$3p4s$”。
2. **中间区域**：顶部居中显示“Si $3pns$”，包含四个垂直列，底部依次标注“$3p4s$”、“$3p5s$”、“$3p6s$”、“$3p7s$”。能级线在 $3p5s$ 和 $3p6s$ 位置呈现过渡状态：左侧能级通过虚线连接中间能级，中间能级再通过虚线连接右侧能级，形成连续演化路径。
3. **右侧区域**：顶部标注“$jj$ 耦合”（中文）和“$jj$ coupling”（英文），垂直排列四个水平能级线（实线），从上至下依次标注为 $(1/2,3/2)_1$、$(1/2,3/2)_2$、$(1/2,1/2)_1$、$(1/2,1/2)_0$，底部对应标签“$3p7s$”。

图中所有能级线均为黑色，连接线使用点划线（左侧到中间）和虚线（中间到右侧）表示耦合过渡。左侧能级间距较大，右侧能级间距较小，直观展示从 $LS$ 耦合到 $jj$ 耦合的能级结构变化。整个图示无彩色元素，仅通过线条样式和位置变化表达物理概念。
