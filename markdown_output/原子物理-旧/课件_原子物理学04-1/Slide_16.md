# Stern-Gerlach实验

- **实验装置**  
  加热炉 → 银原子束 → 准直狭缝 → 非均匀磁场（S极在上，N极在下）→ 探测屏

- **对比图示**  
  - Classical prediction（经典预测）：连续分布的原子束轨迹  
  - What was actually observed（实际观测）：分立的两条原子束轨迹  
  - 标注元素：Silver atoms（银原子）、Inhomogeneous magnetic field（非均匀磁场）、Furnace（加热炉）

- **科学家信息**  
  - Otto Stern (1888 – 1969)  
  - W. Gerlach (1889 – 1979)

- **坐标系示意图**  
  y-z平面中，S极位于上方，N极位于下方，箭头表示原子束在磁场中的偏转方向

## Figure & Layout Description

页面顶部为黑色粗体标题"Stern-Gerlach实验"，下方有一条深灰色水平分割线。主体内容分为三个区域：

1. **左侧实验装置图**（黑白线稿）：
   - 最左侧为立方体"加热炉"，右侧延伸出水平箭头表示"银原子束"
   - 原子束穿过两片垂直平行板构成的"准直狭缝"
   - 进入上下对置的磁极区域：上磁极为"S"标记的梯形块，下磁极为"N"标记的梯形块，标注"非均匀磁场"
   - 右侧为倾斜矩形"探测屏"，显示原子束分裂后的投影

2. **右上区域**：
   - 两张黑白人物肖像照片垂直排列
   - 上方照片标注"Otto Stern (1888 – 1969)"
   - 下方照片标注"W. Gerlach (1889 – 1979)"
   - 肖像左侧有y-z坐标系示意图：垂直z轴箭头向上，水平y轴向右，S极位于坐标原点上方，N极位于下方

3. **底部对比示意图**（彩色增强）：
   - 左侧灰色竖板标注"Classical prediction"，显示连续的浅蓝色光束轨迹
   - 中间磁极区域标注"Inhomogeneous magnetic field"，S/N极用深灰色块表示
   - 右侧蓝色光束分为两条平行轨迹，指向标注"What was actually observed"的探测区域
   - 光束路径标注"Silver atoms"，最右侧立方体标注"Furnace"
