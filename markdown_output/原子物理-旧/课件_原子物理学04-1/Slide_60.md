## 氢原子精细结构能级

### 能级结构标注
- **n=1**  
  $1s$ → $1^2S_{1/2}$  
  能量差：$1.46 \text{ cm}^{-1}$

- **n=2**  
  $2s$ → $2^2S_{1/2}$  
  $2p$ → $2^2P_{1/2}$, $2^2P_{3/2}$  
  能量差：$0.091 \text{ cm}^{-1}$, $0.365 \text{ cm}^{-1}$

- **n=3**  
  $3s$ → $3^2S_{1/2}$, $3^2P_{1/2}$  
  $3p$ → $3^2P_{3/2}$, $3^2D_{3/2}$  
  $3d$ → $3^2D_{5/2}$  
  能量差：$0.036 \text{ cm}^{-1}$, $0.108 \text{ cm}^{-1}$, $0.018 \text{ cm}^{-1}$

### 能量修正分步表达式
| 修正阶段 | 表达式 |
|---------|--------|
| 基础能级 | $E_n$ |
| + 相对论动能修正 | $E_n + E_T$ |
| + 达尔文项 | $E_n + E_T + E_V$ |
| + 自旋-轨道耦合 | $E_n + E_T + E_V + E_{ls}$ |

## Figure & Layout Description
图表以垂直能量轴 $E$ 为纵轴，左侧标注主量子数 $n=1,2,3$。顶部标题为"氢原子精细结构能级"，下方通过四层嵌套结构展示能级分裂过程：
1. **最外层虚线框**：表示未修正的玻尔能级 $E_n$，$n=1$ 为单能级，$n=2$ 和 $n=3$ 显示简并态
2. **第二层虚线**：加入相对论动能修正 $E_T$ 后，$s$ 轨道与 $p/d$ 轨道开始分离
3. **第三层虚线**：加入达尔文项 $E_V$ 后，$s$ 轨道能量进一步上移
4. **最内层实线**：最终加入自旋-轨道耦合 $E_{ls}$，产生精细结构分裂

各分裂能级右侧标注光谱项符号（如 $2^2P_{3/2}$），能量差值用垂直箭头标注具体数值（单位 $\text{cm}^{-1}$）。底部水平排列四个能量表达式，通过虚线连接对应修正阶段。所有线条均为黑色，无填充色，能量差箭头为实心黑色箭头，关键数值用小数点后三位精度标注。
