# 动能的相对论修正(了解)

在类氢离子的本征态下，求其平均值

$$\Delta E_T = -\frac{1}{2m_e c^2}\left\langle (E_n - V(r))^2 \right\rangle$$
$$= -\frac{1}{2m_e c^2}\left\{ E_n^2 + 2E_n \left\langle \frac{Ze^2}{4\pi\varepsilon_0 r} \right\rangle + \left\langle \frac{Z^2 e^4}{(4\pi\varepsilon_0)^2 r^2} \right\rangle \right\}$$

利用 
$$\left\langle \frac{1}{r} \right\rangle = \frac{1}{n^2}\frac{Z}{a_0} \quad \left\langle \frac{1}{r^2} \right\rangle = \frac{1}{(l + \frac{1}{2})n^3}\left( \frac{Z}{a_0} \right)^2$$

得 
$$\Delta E_T = -E_n \frac{\alpha^2 Z^2}{n^2} \left( \frac{3}{4} - \frac{n}{l + \frac{1}{2}} \right)$$

动能的相对论修正造成氢原子(类氢离子)能级的下降
