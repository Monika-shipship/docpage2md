# 氢原子光谱的精细结构

## 能级结构

- **n=3 能级**
  - $3^2D_{5/2}$
  - $3^2P_{3/2}$, $3^2D_{3/2}$
  - $3^2S_{1/2}$, $3^2P_{1/2}$
  - 分裂间隔：$0.018\ \text{cm}^{-1}$, $0.036\ \text{cm}^{-1}$, $0.108\ \text{cm}^{-1}$

- **n=2 能级**
  - $2^2P_{3/2}$
  - $2^2S_{1/2}$, $2^2P_{1/2}$
  - 分裂间隔：$0.091\ \text{cm}^{-1}$, $0.365\ \text{cm}^{-1}$

- **n=1 能级**
  - $1^2S_{1/2}$
  - 分裂间隔：$1.46\ \text{cm}^{-1}$

## 跃迁与光谱

- **n=3 → n=2 跃迁标记**：
  - (a), (b), (c), (d), (e), (f), (g)

- **光谱特征**：
  - 两组谱线：(a)(b)(c) 和 (d)(e)(g)
  - 频率间隔：$\Delta\nu$

## Figure & Layout Description
图片标题"氢原子光谱的精细结构"位于页面顶部中央，采用黑色粗体字，下方有一条深灰色水平分隔线。整个页面分为左右两个主要区域。

左侧区域为能级示意图，垂直方向表示能量E（从下到上能量递增）。图中包含三个主能级：
- n=1能级位于最底部，显示为单条水平线，下方标注"n=1"，右侧标注"1²S₁/₂"，上方有虚线表示精细结构分裂，标注"1.46 cm⁻¹"。
- n=2能级位于中部，显示为多条水平线：最下方是"2²S₁/₂, 2²P₁/₂"，上方是"2²P₃/₂"，分裂间隔分别标注"0.365 cm⁻¹"和"0.091 cm⁻¹"。
- n=3能级位于顶部，显示为多条水平线：最下方是"3²S₁/₂, 3²P₁/₂"，中间是"3²P₃/₂, 3²D₃/₂"，最上方是"3²D₅/₂"，分裂间隔分别标注"0.108 cm⁻¹"、"0.036 cm⁻¹"和"0.018 cm⁻¹"。
- 有一条蓝色粗箭头从n=3能级指向n=2能级，表示电子跃迁。

右侧区域包含两个子图：
- 上方是详细的能级跃迁图，n=3能级在上，n=2能级在下。7条斜向跃迁线分别标记为(a)、(b)、(c)、(d)、(e)、(f)、(g)，连接不同精细结构子能级。
- 下方是光谱强度分布图，横轴为频率ν，纵轴为强度。图中显示两个主要峰，左侧峰对应(a)(b)(c)跃迁，右侧峰对应(d)(e)(g)跃迁，两峰之间有间隔标记"Δν"。

整个图像使用黑色线条和文字，仅跃迁箭头为蓝色。能量间隔值均以"cm⁻¹"为单位精确标注在相应能级之间。
