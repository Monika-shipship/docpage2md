# 总角动量

引入电子的总角动量 $J = L + S$

$J$ 满足角动量量子化关系
$$J^2 = j(j+1)\hbar^2$$
$$J_z = m_j\hbar$$

其中 $j$ 是总角动量量子数，描述其大小；$m_j$ 是相应的磁量子数，描述其z分量。

有
$$\frac{dS}{dt} = \zeta(r)L \times S$$
$$S \times S = 0$$
$$\frac{dL}{dt} = \zeta(r)S \times L$$
$$L \times L = 0$$

$$\frac{dS}{dt} = \zeta(r)(L + S) \times S = \zeta(r)J \times S$$
$$\frac{dL}{dt} = \zeta(r)(S + L) \times L = \zeta(r)J \times L$$

$L$和$S$受到的力矩都垂直于角动量矢量本身，$L$和$S$以相同的角速度 $\omega = \zeta(r)J$ 围绕$J$做拉莫尔进动，大小不变，方向改变。
