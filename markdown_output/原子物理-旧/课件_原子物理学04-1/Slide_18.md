# Stern-Gerlach实验

## 量子力学结果：

空间取向量子化  
$$\mu_{lz} = -\frac{e}{2m_e}m_l\hbar = -m_l\mu_B$$  
$$m_l = -l, -(l-1), \cdots, (l-1), l$$
