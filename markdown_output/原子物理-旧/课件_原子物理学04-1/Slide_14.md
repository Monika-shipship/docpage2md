# 磁矩与磁场的相互作用

磁矩$\boldsymbol{\mu}$在磁场$\boldsymbol{B}$中具有势能(取向势)
$$U = -\boldsymbol{\mu} \cdot \boldsymbol{B} = -(\mu_x B_x + \mu_y B_y + \mu_z B_z)$$

磁矩在磁场中所受到的力
$$\boldsymbol{F} = -\nabla U = -\left( \boldsymbol{i} \frac{\partial U}{\partial x} + \boldsymbol{j} \frac{\partial U}{\partial y} + \boldsymbol{k} \frac{\partial U}{\partial z} \right)$$

其直角坐标分量
$$
\begin{cases}
F_x = -\frac{\partial U}{\partial x} = \mu_x \frac{\partial B_x}{\partial x} + \mu_y \frac{\partial B_y}{\partial x} + \mu_z \frac{\partial B_z}{\partial x} \\
F_y = -\frac{\partial U}{\partial y} = \mu_x \frac{\partial B_x}{\partial y} + \mu_y \frac{\partial B_y}{\partial y} + \mu_z \frac{\partial B_z}{\partial y} \\
F_z = -\frac{\partial U}{\partial z} = \mu_x \frac{\partial B_x}{\partial z} + \mu_y \frac{\partial B_y}{\partial z} + \mu_z \frac{\partial B_z}{\partial z}
\end{cases}
$$

如果磁场是均匀的，梯度为零，则磁矩所受力为零。
