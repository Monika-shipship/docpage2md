# 多电子原子-基态电子组态

$N$个电子组态 $\quad n_1 l_1, n_2 l_2, \cdots n_N l_N$

## 基态电子组态？

(1) 能量最低

体系总能量 $\quad E^{(0)} = \sum_{i=1}^{N} E_{n_i l_i} \quad$ 其中单电子能量 $\quad E_{nl} = -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z_{nl}^{*2}}{n^2}$

$1s,1s,\cdots1s \quad ?$

(2) Pauli不相容原理

支壳层容纳的电子数目受到不相容原理限制

↓

按照支壳层能量的高低，电子由低到高依次排布

## Figure & Layout Description

页面顶部为黑色粗体标题"多电子原子-基态电子组态"，下方有一条灰色水平分割线。标题下方左侧为黑色常规字体"$N$个电子组态"，右侧紧接公式"$n_1 l_1, n_2 l_2, \cdots n_N l_N$"。核心问题"基态电子组态？"以红色加粗字体居中显示。内容分为两个主要部分：  
(1) 以蓝色字体标注的"能量最低"作为一级子标题，其下包含两行公式：左侧为"体系总能量"的黑色说明文字与公式$E^{(0)} = \sum_{i=1}^{N} E_{n_i l_i}$，右侧为"其中单电子能量"的黑色说明文字与公式$E_{nl} = -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z_{nl}^{*2}}{n^2}$。公式下方有红色禁止符号（红色圆圈内加斜线）覆盖的"$1s,1s,\cdots1s$"文字，右侧紧跟红色问号。  
(2) 以蓝色字体标注的"Pauli不相容原理"作为一级子标题，其下为黑色说明文字"支壳层容纳的电子数目受到不相容原理限制"，正下方有一个蓝色向下箭头"↓"，箭头下方为黑色结论文字"按照支壳层能量的高低，电子由低到高依次排布"。  
整体布局为纵向结构，文字与公式按逻辑层级排列，关键概念通过颜色（红/蓝）和符号（禁止标识/箭头）强化视觉层次。
