# 电子的组态 — 顺磁性与抗磁性

- 当轨道中的电子为自旋平行，原子具有顺磁性 (paramagnetism)，  
  例如：氧 ($^8$O): (1s)$^2$(2s)$^2$(2p)$^4$  
  ($↑↓$)$_{1s}$ ($↑↓$)$_{2s}$  
  ($↑↓$)$_{2px}$($↑$)$_{2py}$($↑$)$_{2pz}$

- 当轨道中的电子均为自旋反平行，原子具有抗磁性 (diamagnetism)，  
  例如：铍 ($^4$Be): (1s)$^2$(2s)$^2$  
  ($↑↓$)$_{1s}$ ($↑↓$)$_{2s}$

- 固体的晶体结构会改变其磁性，  
  例如 碳 ($^6$C): (1s)$^2$(2s)$^2$(2p)$^2$  
  ($↑↓$)$_{1s}$ ($↑↓$)$_{2s}$ ($↑$)$_{2px}$($↑$)$_{2py}$  
  原子本身为顺磁性，但热解石墨 (pyrolytic graphite) 具有极强的抗磁性
