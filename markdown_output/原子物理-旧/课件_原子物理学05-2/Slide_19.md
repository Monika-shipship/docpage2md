# 多电子原子-基态电子组态

## 马德隆规则 Madelung's rule

1. Orbitals are filled in the order of increasing $n+l$;
2. Where two orbitals have the same value of $n+l$, they are filled in order of increasing $n$.

页面采用左右分栏布局：

  - 文字左对齐，与右侧图表保持约15%页面宽度的间距  
- **右侧图表区**：  
  - **垂直轴**：左侧有黄色边框的矩形列（高度约页面40%），内含黑色大写字母K、L、M、N、O、P、Q（从下到上排列）

    - 左侧标注主量子数 $n=1$ 至 $n=7$（$n=1$ 在最底部，$n=7$ 在顶部）  
    - 底部标注角量子数 $l=0$（对应s轨道）、$l=1$（p轨道）、$l=2$（d轨道）、$l=3$（f轨道）  
  - **轨道网格**：  
    - $n=1$ 行：仅1s轨道  
    - $n=2$ 行：2s、2p轨道  
    - $n=3$ 行：3s、3p、3d轨道  
    - $n=4$ 行：4s、4p、4d、4f轨道

    - $n=7$ 行：7s轨道  
    - 所有轨道符号以黑色无衬线字体显示在灰色边框方格内

    - 7条红色实线箭头从左下向右上斜向排列  
    - 箭头起点：1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s  
    - 箭头末端带实心三角箭头，指示轨道填充方向
