# 多电子原子-洪特定则和原子基态

原子基态一般遵从$LS$耦合，基态电子组态在$LS$耦合下能级会分裂，那么在这些分裂的原子多重态能级中，哪一个是“真正”的基态呢？

1925年，洪特(F. Hund)提出了一个经验的法则。

**法则一：**

对一给定的电子组态，能量最低的原子态必定具有泡利原理所允许的**最大$S$值**。

F. Hund  
(1896–1997)
