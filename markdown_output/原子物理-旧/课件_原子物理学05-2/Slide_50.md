# 电子组态能级的修正-LS耦合

## (2) LS耦合下多重态能级的精细结构

进一步考虑自旋一轨道相互作用的修正

$$\hat{H} = \hat{H}_0 + \hat{H}_1 + \hat{H}_2$$

$L$ 和 $S$ 不再是守恒量，$M_L$ 和 $M_S$ 不再是好量子数。

引入总角动量

$$J = L + S$$
