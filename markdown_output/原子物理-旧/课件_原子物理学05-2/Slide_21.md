# 多电子原子-元素周期律

## Figure & Layout Description
页面主体由左侧大型元素周期表展示墙和右侧三组垂直排列的元素特写图构成。左侧展示墙采用浅棕色框架，整体呈倾斜视角展示，包含7行18列的网格结构。每个网格单元为透明亚克力材质，内部嵌入蓝色背光照明，部分单元格内可见真实元素样本（如金属块、矿石等）。其中第2周期第18族的Ne（氖）单元格以红色背光高亮显示，内部发光管清晰呈现"Ne"符号及原子序数"10"。展示墙左上角嵌入一块电子显示屏，显示模糊的分子结构图与数据参数。右侧三组特写图均采用白色背景展示盒：最上方为Cr（铬）单元格特写，黑色标签标注"Cr"，内部陈列3块银色金属块与圆形徽章；中间为Mn（锰）特写，标签标注"Mn"，盒内放置灰褐色矿石样本；底部为P（磷）特写，标签标注"P"，展示红色包装盒、透明试管及白色粉末状物质。整体页面采用暖黄色环境光照明，展示墙底部设有线性光源增强立体感。
