# 电子组态能级的修正

## (2) 自旋轨道相互作用

$$\hat{H}_2 = \sum_{i=1}^{N} \zeta(r_i) l_i \cdot s_i$$

对于满支壳层的电子组态
$$m_l = -l, -l+1, \cdots, +l-1, +l$$
$$m_s \pm 1/2 \quad \cdots \quad \pm 1/2$$

有 $$M_L = \sum m_l = 0 \quad M_S = \sum m_s = 0$$

从而 $$\boldsymbol{L} = \boldsymbol{S} = \boldsymbol{0}$$

同样，只要计及最外未满支壳层上的价电子即可。

类H离子自旋轨道相互作用能：
$$\Delta E_{ls} = \frac{1}{2m^2c^2} \left( \frac{Ze^2}{4\pi\epsilon_0} \right) \frac{Z^3}{a_0^3 n^3 l(l+\frac{1}{2})(l+1)}$$

$$\hat{H}_2$$ 大致与$Z^4$成比
