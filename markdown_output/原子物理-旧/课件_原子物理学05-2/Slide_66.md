## 电子组态能级的修正-LS耦合

等效电子组态 $nd^2$

$l_1 = l_2 = 2$ $\longrightarrow$ $L = 0, 1, 2, 3, 4$

$s_1 = s_2 = 1/2$ $\longrightarrow$ $S = 0, 1$

$L+S = \text{偶数}$

$\longrightarrow$ $S = 0$; $L = 0, 2, 4$  
原子多重态：$^1G_4$, $^1D_2$, $^1S_0$,

$S = 1$; $L = 1, 3$  
原子多重态：$^3P_{2,1,0}$, $^3F_{4,3,2}$

   - 下分支显示"$S = 1$; $L = 1, 3$"，其下方缩进显示"原子多重态："及两个光谱符号  
所有文本使用黑色无衬线字体，光谱符号中上标数字（如$^1$）为较小字号，下标字母（如$G_4$）为标准字号。页面背景为纯白色，无其他装饰元素。
