# 多电子原子—中心力场近似与电子组态

考察一个原子或离子，由一个带$+Ze$电荷的原子核和核外$N$个电子组成。

(1) 设原子核质量无穷大，将原子核取为坐标原点；  
(2) 考虑非相对论的情形。

总波函数  
$$\psi(q_1, q_2, \cdots q_N) = u(r_1, r_2, \cdots, r_N) \chi(s_1, s_2, \cdots, s_2)$$  
空间波函数 $\quad$ 自旋波函数
