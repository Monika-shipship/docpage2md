# 电子组态能级的修正-LS耦合

## (3) 朗德间隔定则

在$LS$耦合下，自旋—轨道相互作用项可以写作

$$\hat{H}_2 = \xi(L,S) L \cdot S$$

$\xi(L,S)$ 是与电子组态和$L$、$S$量子数有关的一个常数。

$$L \cdot S = \frac{1}{2}(J^2 - L^2 - S^2)$$

精细结构修正：

$$\Delta E_J = \frac{\hbar^2}{2} \xi(L,S) [J(J+1) - L(L+1) - S(S+1)]$$

同—多重态相邻能级的间隔为

$$E_J - E_{J-1} = \frac{\hbar^2}{2} \xi(L,S) [J(J+1) - (J-1)J] = \xi(L,S) \hbar^2 J$$
