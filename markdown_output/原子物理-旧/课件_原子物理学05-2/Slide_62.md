# Slide 62

## 电子组态能级的修正-LS耦合

Slater图解法求$np^2$电子组态的谱项

$$L=2, S=0; \, ^1D$$

## Figure & Layout Description

页面包含两个并列的浅蓝色圆角矩形图表，由深蓝色右向箭头连接。左侧图表为$M_L$-$M_S$二维分布图：  
- 纵轴标注$M_L$（取值-2,-1,0,1,2），横轴标注$M_S$（取值-1,0,1）  
- 坐标网格内分布白色圆形节点，节点内数字表示允许态数量：  
  - $M_L=0$轴线上：$M_S=-1$和$M_S=1$处为"1"，$M_S=0$处为"3"  
  - $M_L=\pm1$轴线上：$M_S=0$处为"2"，$M_S=\pm1$处为"1"  
  - $M_L=\pm2$轴线上：仅$M_S=0$处为"1"  
- 背景色为浅蓝色（#B0D0E0），坐标轴为黑色细线  

右侧图表为简化分布图：  
- 仅保留$M_S=0$轴线上的5个"1"节点，沿$M_L$轴从-2到+2对称分布  
- 无其他坐标刻度，仅保留$M_L$和$M_S$轴标签  
- 底部标注黑色公式$L=2, S=0; \, ^1D$  

整体布局为标题-说明文字-双图表结构，标题下方有深灰色分隔线，图表间箭头表示推导过程。
