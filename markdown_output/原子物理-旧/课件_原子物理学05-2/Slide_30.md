# 多电子原子-元素周期律

## 第七周期

从钫(Fr: Francium, $Z = 87$)到118号(未命名)，依次填充$7s$, $5f$, $6d$, $7p$支壳层。

从钍(Th: Thorium, $Z = 90$)起到铹(Lr: Lawrencium, $Z = 103$) 新增加的电子不断填充 $5f$ 支壳层，这些元素形成**锕系元素**(包括锕，Actinium)。

自然界存在的元素到铀($Z = 92$)为止，比铀更重的元素都是人工合成的。

## Figure & Layout Description

页面顶部是黑色粗体主标题"多电子原子-元素周期律"，下方有深蓝色分隔线。标题下方左侧有红色二级标题"第七周期"。正文包含三段黑色文字，第一段描述第七周期元素填充顺序，第二段说明锕系元素形成机制（"锕系元素"用蓝色突出显示），第三段指出自然界元素的原子序数上限。

页面下半部分包含两个水平排列的表格：
1. 上层表格展示第七周期主表，包含18列元素（87-118号），其中89-103号元素区域用橙色背景填充并标注"actinoids"，该区域外围有黄色粗线框，框下方标注"$6d^1 7s^2$"，并有一条蓝色向下箭头指向下方表格。
2. 下层表格为橙色背景的锕系元素特写表（90-103号），每格包含元素符号、英文名称和原子量，表格底部用红色字体标注各元素的电子组态。

整体布局为典型教学PPT结构：标题→理论说明→图表验证。关键概念通过颜色编码强化：红色表示周期序号，蓝色表示重点术语，橙色表示锕系元素区域，黄色框突出核心区域，蓝色箭头指示逻辑关联。
