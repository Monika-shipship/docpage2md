# 多电子原子-原子的壳层结构

$N$个电子组态 $\quad n_1 l_1, n_2 l_2, \cdots n_N l_N$

单电子轨道半径(Bohr图像)
$$r_{nl} = a_0 \frac{n^2}{Z_{nl}^{*2}}$$

处在相同 $nl$ 状态的电子：轨道半径相同，能量相同，  
称为处在同一支壳层。

$l = 0, 1, 2, 3, 4, 5, \ldots$ 分别称为：s, p, d, f, g, h …支壳层。

处在相同 主量子数 $n$ 状态的电子：轨道半径相近，能量相近，  
称为处在同一壳层。

$n = 1, 2, 3, 4, 5, 6, 7$ 分别称为：K, L, M, N, O, P, Q 壳层。
