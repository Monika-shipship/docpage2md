# Slide 96

液态氧具有順磁性

## Figure & Layout Description
图片采用居中对称布局，整体背景为均匀的浅灰色调，营造出实验室环境的简洁感。顶部有一行黑色中文标题文字“液态氧具有順磁性”，字体为标准无衬线体（类似黑体），字号较大，居中对齐，文字下方紧接一条细长的深蓝色水平分割线（宽度约等于标题文字长度）。主体视觉内容位于分割线下方，占据图片大部分区域：左右两侧各有一个金属材质的锥形磁极，呈银灰色，表面光滑且有明显高光反射，锥体从底部宽端向中心尖端逐渐收窄，尖端相对并保持约1-2厘米间隙。在磁极间隙中央，可见一束透明的液态氧被磁化后形成的细长桥状结构，呈半透明浅蓝色，连接两侧磁极尖端，其形态在磁场作用下略微拉伸并保持稳定。图片右下角有模糊的白色手写体文字“0.5T”，疑似表示磁场强度为0.5特斯拉，但因图像分辨率较低，部分笔画略显模糊。整体色彩以冷色调为主（灰、银、蓝），突出科学实验的客观性；视觉焦点集中在液态氧的桥状结构上，通过金属磁极的几何对称性和液态氧的透明质感，直观展示磁场对顺磁性物质的作用效果。
