# 多电子原子-元素周期律

在总结元素的化学和物理性质的基础上，1869年门捷列夫(D.I. Mendeleev)发现了元素周期律。

把元素按原子量大小的次序排列起来，发现元素性质呈现出周期变化。

D. Mendeleev (1834-1907)

在原子序数$Z = 2$, $10$, $18$, $36$等惰性元素处，电离能呈现峰值，而在$Z = 3$, $11$, $19$等碱金属元素处，电离能呈现极小值。

"幻数"
