## Figure & Layout Description
页面顶部有黑色粗体标题"多电子原子-元素周期律"，下方有深灰色分隔线。主体部分为IUPAC标准元素周期表，采用网格布局：  
1. 表格结构：主表包含7个周期（行）和18个族（列），最右侧列标有1-18族编号  
2. 元素单元格：每个格子包含4层信息（从上至下）：原子序数（如"1"）、元素符号（如"H"）、元素名称（如"hydrogen"）、标准原子量（如"[1.0078, 1.0082]"）  
3. 特殊区域：  
   - 第6-7周期下方有橙色背景区域标注"lanthanoids"（57-71）和"actinoids"（89-103）  
   - 左下角有IUPAC标志（地球仪图案）及彩色条（红/橙/黄/绿/青/蓝/紫）  
4. 文字特征：  
   - 表格上方有英文标题"IUPAC Periodic Table of the Elements"  
   - 右下角有版权说明文字（2016年版本）  
   - 所有元素符号均使用标准化学符号格式（如"Na"、"Fe"等）  
5. 颜色系统：  
   - 主表格为白色背景，黑色文字  
   - 镧系/锕系区域为浅橙色背景  
   - 族编号区域为浅灰色背景
