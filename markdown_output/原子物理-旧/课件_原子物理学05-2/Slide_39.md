# 电子组态能级

对于满支壳层的电子组态($v = 2(2l+1)$)，$G = 1$

$$m_l = -l, -l+1, \cdots, +l-1, +l$$

$m_s$  $ \pm 1/2 $  $\cdots$  $ \pm 1/2 $

有

$$M_L = \sum m_l = 0$$

$$M_S = \sum m_s = 0$$

从而 $L = S = 0$
