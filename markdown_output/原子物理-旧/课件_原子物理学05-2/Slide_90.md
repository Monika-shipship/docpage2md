# 多电子原子光谱-电偶极跃迁的选择定则

单电子状态的宇称为 $(-1)^l$  

若 $l$ 为偶数，则波函数在反演变换下不变，称该状态具有偶宇称。  

若 $l$ 为奇数，则波函数在反演变换下变号，称该状态具有奇宇称。  

对于多电子原子，在中心力场近似下，原子的状态由电子组态描述，宇称为：  
$$(-1)^{\sum l_i}$$  

在电偶极辐射中，光子带走的角动量为 $\hbar$，要求电偶极跃迁的选择定则是  
$$\Delta \sum l_i = \pm 1$$
