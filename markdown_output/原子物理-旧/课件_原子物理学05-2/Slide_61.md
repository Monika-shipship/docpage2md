# 电子组态能级的修正-LS耦合
Slater图解法求np²电子组态的谱项

## Figure & Layout Description
PPT页面顶部是主标题"电子组态能级的修正-LS耦合"，采用黑色粗体字（约32pt），下方有一条深灰色水平分隔线（粗细约2px）。分隔线下方是副标题"Slater图解法求np²电子组态的谱项"，采用黑色常规字体（约18pt）。页面主体区域左侧包含一个浅蓝色圆角矩形图表（圆角半径约10px，背景色#C6E2EE），图表内部构建了二维坐标系：垂直轴（Y轴）标记为$M_L$，向上箭头指示正方向，轴左侧刻度从下至上依次标注-2、-1、0、1、2；水平轴（X轴）标记为$M_S$，向右箭头指示正方向，轴底部刻度从左至右依次标注-1、0、1。坐标系网格点上分布着13个白色圆形节点（直径约15px，黑色边框），节点内有黑色阿拉伯数字：中心位置$(M_L=0, M_S=0)$为3；上下对称位置$(M_L=1, M_S=0)$和$(M_L=-1, M_S=0)$均为2；最外层对称位置$(M_L=2, M_S=0)$、$(M_L=-2, M_S=0)$、$(M_L=0, M_S=1)$、$(M_L=0, M_S=-1)$、$(M_L=1, M_S=1)$、$(M_L=1, M_S=-1)$、$(M_L=-1, M_S=1)$、$(M_L=-1, M_S=-1)$均为1。所有坐标轴线和节点连线均为黑色实线（粗细1px），图表整体布局对称，数字清晰可辨，背景为浅蓝色填充区域。
