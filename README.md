# AI 习题助手 (AI-Exercise-Solver) 🚀

> 基于 **Qwen3-VL-Plus** (通义千问视觉大模型) 的智能习题识别与解析工具。
> 支持 **精准题目提取**、**公式 LaTeX 转换**、**智能解题** 以及 **多任务并发**。

## 📖 简介

本项目旨在将大量的习题图片（如试卷截图、作业照片）批量转换为结构清晰的 Markdown 笔记，并提供详细的解析和答案。

不同于普通的 OCR 工具，本项目利用了大模型的**视觉推理能力**：

1. **精准识别**：自动识别题目文本、选项、小问，并保留原有排版结构。
2. **公式转换**：自动识别数学/物理公式并转换为标准 LaTeX 格式 (`$$...$$`)。
3. **智能解析**：扮演“学科辅导老师”，提供详细的解题思路、步骤和最终答案。
4. **图示描述**：自动分析题目中的几何图、函数图、电路图等，并生成详细的文字描述。
5. **高效并发**：支持同时处理多个文件夹，内置 QPS/TPM 保护机制。

---

## ✨ 核心特性

* **📝 题目与解析分离**：输出结构清晰，包含“题目”、“答案”、“解析”三个独立部分，方便核对和学习。
* **🗝️ 独立答案项**：在题目下方单独列出最终答案，便于快速自查。
* **💻 沉浸式终端 UI**：使用固定进度条显示多任务状态，包含百分比、速度 (s/页)、预计剩余时间 (ETA)。
* **📂 自动分组与隔离**：自动扫描 `ppt_images` 下的子文件夹，输出结果自动隔离到 `markdown_output` 对应目录。
* **⚡ 智能并发控制**：支持同时处理多个任务文件夹，充分利用 API 配额。
* **🧠 深度思考模式**：启用 `Thinking Mode`，极大提升对复杂题目和图形的理解能力。
* **🎮 交互式控制**：启动时自动扫描任务，支持选择特定文件夹、指定页码范围（如 `1-10`, `5-end`）。
* **🔄 断点续传**：自动跳过已生成的文件，无缝接续生成。
* **📑 自动合并**：单页生成完毕后，自动汇总为 `XXX_FULL.md`。

## 🛠️ 环境准备

### 1. Python 环境

需要 Python 3.8 或更高版本。

### 2. 安装依赖

本项目依赖阿里云官方 SDK 和 UI 库 `rich`。

```bash
pip install -U dashscope rich
```

### 3. 配置 API Key

你需要开通阿里云百炼服务并获取 `Qwen3-VL-Plus` 的权限。

*方式 A (推荐 - 环境变量)*:

```powershell
# Windows PowerShell
$env:DASHSCOPE_API_KEY="sk-你的密钥"

# Linux / macOS
export DASHSCOPE_API_KEY="sk-你的密钥"
```

*方式 B (修改代码)*:
打开 `ppt2md.py`，修改顶部配置区：

```python
# dashscope.api_key = "sk-你的密钥"  <-- 去掉注释填入
```

## 🚀 使用指南

### 1. 启动程序

在终端运行：

```bash
python ppt2md.py
```

### 2. 交互式选择

程序启动后会扫描 `ppt_images` 下的所有子文件夹，并进入交互模式：

```text
🔍 正在扫描习题文件夹...

📂 发现以下任务:
  [1] 习题集_第一章 (80 页)
  [2] 物理试卷_期末 (20 页)

👉 请选择要处理的任务:
   输入序号(如 1,3) | 输入 'a' 全选
   您的选择: 
```

### 3. 配置页码范围

选择任务后，你可以指定处理范围（支持断点续传逻辑）：

* `all` 或 `回车`：处理所有图片。
* `1-10`：只处理第 1 到第 10 张图片。
* `20-end`：从第 20 张图片处理到最后。

### 4. 输出示例

生成的 Markdown 文件将包含：

```markdown
# 习题 1

## 📝 题目
如图所示，一个质量为 $m$ 的小球...

## 🗝️ 答案
**B**

## 💡 解析
1. 对小球进行受力分析...
2. 根据牛顿第二定律...
3. 解得...
```
