## 3. 独有功能：Search Agent (联网搜索增强)

`qwen3-max` 与 `qwen3-max-2025-09-23` 模型原生支持 **Search Agent**（联网搜索 Agent 策略）。

### 3.1 功能描述

普通的联网搜索通常是一次性检索，而 **Search Agent** 策略允许模型多次调用联网搜索工具，进行多轮信息检索与内容整合。这非常适合需要高精度、多源交叉验证的研究或报告生成场景。

* **启用方式：** 在 API 调用中设置 `search_strategy` 为 `agent`。
* **适用模型：** 仅 `qwen3-max` 与 `qwen3-max-2025-09-23`。
* **限制：** 启用该策略时，仅支持返回搜索来源（`enable_source: true`），其他联网搜索功能不可用。

### 3.2 额外费用

使用 `agent` 策略进行搜索时，除了模型本身的 Token 费用外，每次调用还会产生额外的服务费：

* **中国大陆 (北京)：** 4 元 / 1000 次调用
* **国际 (新加坡)：** 73.392381 元 / 1000 次调用

> **普通联网搜索费用：** 如果使用默认的 `turbo` 或 `max` 搜索策略，目前是限时免费的（仅收取模型 Token 费）。
